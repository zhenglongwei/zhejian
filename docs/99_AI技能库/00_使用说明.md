# AI 技能库使用说明

本目录存放**可复用的开发工作流提示词**，供 Cursor / AI 助手在专项任务中按项目规范执行，**不是业务代码**。

与 `.cursor/rules/` 的分工：

| 位置 | 作用 |
|---|---|
| `.cursor/rules/` | 长期强约束，每次对话自动加载 |
| `docs/99_AI技能库/` | 专项任务，手动 @ 或明确要求使用 |
| `.cursor/skills/` | Agent 可发现的技能入口，指向本目录对应文档 |

---

## 技能清单

| 文件 | 用途 | 适合时机 |
|---|---|---|
| [skill_design_system_check.md](./skill_design_system_check.md) | 检查是否符合设计体系 | 改完页面后 |
| [skill_tokens_sync.md](./skill_tokens_sync.md) | 同步 token 文档与 `styles/tokens.wxss` | 改颜色/字号/间距时 |
| [skill_component_api_implementation.md](./skill_component_api_implementation.md) | 按 API 规范实现组件 | 新增基础/业务组件 |
| [skill_page_scaffold.md](./skill_page_scaffold.md) | 生成页面骨架 | 新增页面 |
| [skill_wxss_refactor.md](./skill_wxss_refactor.md) | 硬编码样式 → token | 老页面重构 |
| [skill_price_compliance_check.md](./skill_price_compliance_check.md) | 价格/事故车/历史案例合规 | 涉及价格展示 |
| [skill_privacy_desensitization_check.md](./skill_privacy_desensitization_check.md) | 隐私/脱敏/公开逻辑 | 相册/案例/分享 |
| [skill_component_usage_review.md](./skill_component_usage_review.md) | 是否重复造组件 | 页面开发完成后 |
| [skill_api_integration.md](./skill_api_integration.md) | 接口与 loading/empty/error | 接 API 时 |
| [skill_miniprogram_image_debug.md](./skill_miniprogram_image_debug.md) | 小程序图片不显示分层排查 | 封面空白、同 URL 跨页不一致、image-placeholder 有图无显 |
| [skill_release_checklist.md](./skill_release_checklist.md) | 上线前总检 | 发版前 |

**建议优先落地**：`page_scaffold`、`component_api_implementation`、`wxss_refactor`、`price_compliance_check`。

## 使用方式

### 方式一：在对话中指定技能文件

```
请使用 docs/99_AI技能库/skill_design_system_check.md 中的规则，
检查 pages/service/detail 页面。

要求：
1. 只输出问题清单
2. 不要直接修改代码
3. 每个问题给出建议替换方案
```

### 方式二：先读规范再执行

```
请先阅读：
- docs/00_设计规范/00_辙见平台设计体系.md
- docs/00_设计规范/01_设计令牌_tokens.md
- docs/00_设计规范/02_组件API规范.md
- docs/99_AI技能库/skill_page_scaffold.md

然后生成 pages/service/detail 页面骨架。
先输出文件变更计划，等我确认后再写代码。
```

### 方式三：固定开发流程（推荐）

每开发一个页面可按序执行：

1. `skill_page_scaffold`
2. `skill_api_integration`
3. `skill_component_usage_review`
4. `skill_design_system_check`
5. `skill_price_compliance_check` / `skill_privacy_desensitization_check`（按页面类型）
6. `skill_release_checklist`（发版前）

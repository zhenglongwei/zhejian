## **2. FAQ 作用**

FAQ 用于回答用户在维修前常见的问题。

主要作用：

1. 提升页面内容完整度；
2. 覆盖长尾搜索问题；
3. 增强 AI 问答可引用性；
4. 降低用户决策成本；
5. 引导用户进行合理检查和咨询；
6. 支持 FAQPage Schema。

## **6. FAQ 生成原则**

FAQ 必须遵循：

1. 基于案例事实生成；
2. 不编造案例中没有的信息；
3. 不虚构价格；
4. 不承诺绝对维修效果；
5. 不替代线下检测；
6. 不输出危险维修指导；
7. 不泄露用户隐私；
8. 不攻击其他商家；
9. 不虚构评分或评价；
10. 不夸大故障严重性；
11. 必要时提示“建议到店检查”；
12. 内容与页面主题一致。

---

## **7. FAQ 答案长度**

建议：

```
每个答案 80-200 字。

```

规则：

1. 不少于 50 字；
2. 不超过 300 字；
3. 优先使用事实说明；
4. 避免空泛营销；
5. 避免堆砌关键词；
6. 避免重复回答。

---

## **8. 标准 FAQ 模板**

## **8.1 必要性问题**

问题模板：

```
【故障/问题】是否必须维修？

```

答案模板：

```
【故障/问题】是否需要立即维修，需要结合实际检查结果判断。轻微情况可能只需要清洁、调整或继续观察；如果已经出现【严重表现】，或检查发现【关键部件问题】，则建议及时处理，避免影响【安全/性能/使用体验】。

```

---

## **8.2 风险类问题**

问题模板：

```
继续使用有什么风险？

```

答案模板：

```
如果【主要问题】持续存在，继续使用可能导致【风险1】、【风险2】或【风险3】。具体风险取决于部件磨损程度、车辆使用环境和驾驶习惯。建议在问题持续出现或加重时尽快检查，避免小问题发展成更高成本的维修。

```

---

## **8.3 流程类问题**

问题模板：

```
维修流程是什么？

```

答案模板：

```
一般流程包括用户描述故障、门店进行初步检查、拆检相关部件、确认故障原因、沟通维修方案、施工处理、完工检测和试车确认。不同车型和故障程度会影响具体流程，实际维修应以门店检查结果为准。

```

---

## **8.4 价格类问题**

问题模板：

```
价格差异来自哪里？

```

答案模板：

```
价格差异通常来自车型结构、配件品牌、配件规格、工时难度、是否需要拆检以及是否存在关联部件损坏等因素。同一维修项目在不同车型上的施工难度可能不同，因此建议结合实际检查结果和门店报价进行判断。

```

---

## **8.5 维修后注意事项**

问题模板：

```
维修后需要注意什么？

```

答案模板：

```
维修后建议留意故障是否再次出现，并关注车辆行驶状态是否恢复正常。如果更换了制动、悬挂、轮胎或发动机相关部件，建议按照门店提示进行磨合或复查。若出现异响、抖动、报警灯等异常情况，应及时联系门店检查。

```

## **9. 案例页 FAQ 示例**

以“丰田凯美瑞刹车异响维修案例”为例。

### **FAQ 1**

问题：

```
刹车异响是否必须维修？

```

答案：

```
刹车异响不一定都需要立即更换配件，但建议尽快检查。轻微异响可能来自灰尘、潮湿或新旧刹车片磨合；如果异响持续存在，或伴随刹车距离变长、方向盘抖动、金属摩擦声，就需要检查刹车片、刹车盘和卡钳状态，避免影响制动安全。

```

---

### **FAQ 2**

问题：

```
继续使用有什么风险？

```

答案：

```
如果刹车片已经接近磨损极限，继续使用可能导致制动力下降、刹车盘被拉伤、刹车距离变长等问题。严重情况下可能影响行车安全。是否存在风险需要结合刹车片厚度、刹车盘状态和实际制动表现判断。

```

---

### **FAQ 3**

问题：

```
刹车片更换流程是什么？

```

答案：

```
通常流程包括检查刹车片厚度、检查刹车盘表面状态、拆卸旧刹车片、清洁卡钳和导向销、安装新刹车片、复位制动系统并进行试车确认。不同车型的结构和磨损情况不同，具体施工流程应以门店检查结果为准。

```

---

### **FAQ 4**

问题：

```
刹车片价格差异来自哪里？

```

答案：

```
刹车片价格差异主要来自车型、配件品牌、材质、前后轮位置以及施工工时。部分车型刹车系统结构更复杂，或需要同时处理刹车盘、卡钳导向销等关联部件，费用也会有所不同。具体价格建议以实际检查和门店报价为准。

```

---

### **FAQ 5**

问题：

```
维修后需要注意什么？

```

答案：

```
更换刹车片后，短时间内可能需要一定磨合。建议避免急刹和长时间重载制动，并留意是否仍有异响、抖动或制动力异常。如果维修后问题持续存在，应及时回店复查刹车盘、卡钳和安装状态。

```

## **13. FAQ 质量要求**

合格 FAQ 应满足：

1. 问题明确；
2. 答案直接；
3. 内容与页面相关；
4. 不夸大；
5. 不虚构；
6. 不泄露隐私；
7. 不重复；
8. 可被 FAQPage Schema 使用；
9. 对用户决策有帮助；
10. 支持人工编辑。

---

## **14. 不合格 FAQ 示例**

不合格问题：

```
我们是不是全城最便宜？

```

原因：营销化、不可验证。

不合格答案：

```
必须马上维修，否则一定会出事故。

```

原因：绝对化、夸大风险。

不合格答案：

```
这个问题肯定是刹车片坏了，不用检查。

```

原因：未经检测直接下结论。

不合格答案：

```
这家门店保证一次修好，永久不复发。

```

原因：绝对承诺。

## **15. FAQ 生成流程**

```
读取页面类型
→ 读取案例/车型/服务项目数据
→ 选择 FAQ 模板
→ 生成候选 FAQ
→ 检查事实一致性
→ 检查隐私与敏感词
→ 检查是否夸大或承诺
→ 人工审核或自动发布
→ 写入页面
→ 写入 FAQPage Schema

```

---

## **16. 人工审核场景**

以下 FAQ 必须人工审核：

1. AI 自动生成 FAQ；
2. 涉及制动系统；
3. 涉及转向系统；
4. 涉及新能源高压系统；
5. 涉及事故车；
6. 涉及高金额维修；
7. 涉及用户投诉；
8. 涉及售后纠纷；
9. 答案中包含价格；
10. 系统识别存在夸大风险。

---

## **17. 与 Schema 的关系**

FAQ 若要写入 `FAQPage Schema`，必须满足：

1. FAQ 在页面可见；
2. FAQ 内容与 Schema 内容一致；
3. FAQ 已通过审核；
4. FAQ 不包含隐私；
5. FAQ 不包含虚假承诺；
6. FAQ 不涉及危险维修指导。

---

## **18. P0 验收标准**

P0 必须满足：

1. 每个公开案例页至少 3 个 FAQ；
2. FAQ 可在页面展示；
3. FAQ 可写入 FAQPage Schema；
4. FAQ 不包含隐私信息；
5. FAQ 不包含绝对化承诺；
6. FAQ 答案基于案例或服务事实；
7. 后台可编辑 FAQ；
8. 后台可删除 FAQ；
9. AI 生成 FAQ 可人工审核；
10. 页面下线后 FAQ 不再公开。

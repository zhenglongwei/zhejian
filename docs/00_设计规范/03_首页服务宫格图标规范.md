# 首页 · 核心服务宫格图标规范（设计师交付）

> **用途**：供 VI/插画设计师输出首页 6 宫格定制图标，替换 `assets/home/entries/*.png`。  
> **业务口径**：`docs/02_用户端小程序/02_首页与服务发现.md` §10.1.1  
> **页面实现**：`pages/home/index` · `constants/home-entries.js` · 图标环样式见 `pages/home/index.wxss`

---

## 1. 设计定位

| 项目 | 要求 |
| --- | --- |
| 气质 | 专业、克制、可信赖；**服务相册平台**，非电商促销 |
| 风格 | **统一线面图标**（推荐 2px 描边 + 可选轻填充），全套语义、粗细、圆角一致 |
| 禁止 | 红包/金币、「最低价」角标、夸张 3D、霓虹渐变、各入口风格混搭 |
| 事故车 | 可警示语义，但**不得**暗示线上报价（避免「¥」「计算器」「一口价」） |

---

## 2. 交付清单（6 项）

| 文件名 | 服务名称 | 语义建议 | 页面 accent |
| --- | --- | --- | --- |
| `entry_brake.png` | 刹车片/盘 | 刹车盘 + 卡钳弧线 | primary 蓝 |
| `entry_tire.png` | 轮胎服务 | 轮胎断面 / 轮毂 | primary 蓝 |
| `entry_battery.png` | 电瓶更换 | 蓄电池正负极 | info 青 |
| `entry_body.png` | 钣喷修复 | 车身 + 喷漆/修复 | onsite 橙 |
| `entry_accident.png` | 事故车维修 | 警示三角 +「检测」感（非碰撞血腥） | accident 紫 |
| `entry_maintenance.png` | 小保养 | 机油壶 / 滤芯 / 扳手 | success 绿 |

**源文件（必交）**：`assets/home/entries-src/entry_*.svg`（可编辑矢量）  
**上线文件（必交）**：`assets/home/entries/entry_*.png`

---

## 3. 尺寸与导出

| 项 | 规格 |
| --- | --- |
| 画布 | **96 × 96 px**（逻辑尺寸，对应小程序 `88rpx` 图标区） |
| 导出 | **192 × 192 px** PNG（@2x），或 **288 × 288**（@3x 可选） |
| 背景 | **透明**（禁止在 PNG 内铺底色块；底色由页面 `icon-ring` 渐变承担） |
| 安全区 | 图形主体距边 **≥ 12px**（@1x），避免贴边裁切 |
| 色彩 | 主图形 `#1677FF`（`--color-primary`）；**不要**在图标内写服务名汉字 |

### 3.1 推荐导出方式

**方式 A · Figma / Sketch**

1. 打开项目内参考稿：`assets/home/entries-src/*.svg`
2. 统一 Frame 96×96，图标 72×72 居中
3. Export → PNG @2x → 覆盖 `assets/home/entries/`

**方式 B · 命令行（已装 Cairo）**

```bash
pip install cairosvg
python scripts/export-home-entry-icons.py
```

## 4. 视觉参考（与页面对齐）

```
┌─────────────────────────────────────┐
│ ▬▬▬ 顶部渐变强调条（页面 CSS）        │
│ 核心服务                             │
│ 选择服务类型，查看案例或预约咨询      │
│ ┌────┐ ┌────┐ ┌────┐               │
│ │ ○  │ │ ○  │ │ ○  │  ← 圆形渐变环  │
│ │图标│ │图标│ │图标│  ← 本规范交付   │
│ │名称│ │名称│ │名称│               │
│ └────┘ └────┘ └────┘               │
└─────────────────────────────────────┘
```

- 图标仅负责**图形识别**；标签「参考区间 / 到店检测 / 热门」由 `ui-tag` 渲染，**不要画进图标**。
- 与 TabBar 图标、案例封面图**分开设计**；宫格图标可略饱满，但仍需线面统一。

## 5. 品牌色（须严格）

| Token | 色值 | 用途 |
| --- | --- | --- |
| `--color-primary` | `#1677FF` | 默认 4 项图标描边/填充 |
| `--color-info` | `#14C9C9` | 电瓶（可选与 primary 二选一，建议仍用 primary 描边） |
| 页面环背景 | 见 `tokens.wxss` | **不在 PNG 内实现** |

> 全套图标建议**统一用 primary 描边**；语义色仅体现在页面圆环渐变，避免 6 个图标 6 种颜色显得杂乱。

## 6. 验收标准

- [ ] 6 个 PNG 文件名与上表一致，透明底，@2x 清晰
- [ ] 6 个 SVG 源文件已提交至 `entries-src/`
- [ ] 线宽、圆角、视觉重量一致；缩小至 44px 仍可辨认
- [ ] 无促销、无价格、无「最低价」视觉
- [ ] 事故车图标无血腥碰撞、无线上报价暗示
- [ ] 覆盖后微信开发者工具预览首页宫格无拉伸、无锯齿

---

## 7. 工程接入（设计师无需改代码）

1. 将 PNG 放入 `assets/home/entries/`
2. 确认 `constants/home-entries.js` 中 `iconPath` 仍为 `/assets/home/entries/entry_*.png`
3. 重新编译小程序 → 预览首页「核心服务」

如需改入口 ID 或增删入口，需同步改 `HOME_SERVICE_ENTRIES` 与本文档 §2 表格。

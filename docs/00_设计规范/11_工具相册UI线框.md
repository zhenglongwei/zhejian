## 0. 信息架构总览（A-06b）

**核心分工**：档案元信息在 **列表卡片**；详情页 **只翻图**；说明与隐私在 **信息抽屉**；授权/分享在 **卡片右上角 + 尾页**；合规与联系在 **页底留白栏**。

| 信息/操作 | 列表卡片 `album/list` | 详情翻页 `album/detail` | 信息抽屉 | 尾页（最后一页） | 页底留白栏 |
| --- | --- | --- | --- | --- | --- |
| 标题/服务/门店/车辆/张数 | ✅ 合并展示 | — | — | — | — |
| 状态 Tag（完工/审核/公开） | ✅ | — | — | — | — |
| 分享 | ✅ 右上角 | — | — | ✅ 按钮 | — |
| 授权公示 | ✅ 右上角（与状态合并） | — | — | ✅ 按状态显示 | — |
| 翻页大图 + 节点短文案 | — | ✅ 相框内 | — | — | — |
| 隐私说明 | — | — | ✅ | — | — |
| 门店对图片/节点说明 | — | — | ✅ 全文 | — | — |
| 节点 Tab 跳转 | — | ✅ 底栏 | — | — | — |
| 反馈 | — | — | — | ✅ 按钮 | — |
| 联系门店 | — | — | — | — | ✅ |
| ComplianceNotice | — | — | — | — | ✅ |

**合并规则**

- **标题区 + 摘要表** → 卡片右侧 **档案信息块**（不再在详情/抽屉重复四行表）。
- **公示状态 + 授权按钮** → 卡片右上角 **单一「授权/公示」入口**（文案随 `publicCaseStatus` 变化）。
- **分享** 与授权同排右上角；详情框内顶栏 **不再放分享**。

## 2. 相册列表 · 加高档案卡片（`pages/album/list`）

> **A-06b**：列表仍是 **可纵向滚动的多卡片列表**（一屏通常 **2～4 条**，非一屏一本）。  
> **A-06c（澄清）**：在现有 **横向卡片** 上 **加高、充实、美化**——左侧 **较大缩略图预览**（首图/封面），右侧档案信息；**不是**整屏竖版「画册封面」。

### 2.1 列表线框

```text
┌──────────────────────────────────────────┐
│ ░ 服务相册 · 门店维修记录          band ░ │
├──────────────────────────────────────────┤
│  [ 私密相册 ]  [ 公开相册 ]   Tabs        │
├──────────────────────────────────────────┤
│ ┌────────────────────────────────────┐ │  ← 单条卡片（加高约 1.5～2× 现卡）
│ │ ┌──────────┐           分享  授权  │ │  右上：分享 + 授权/公示 §2.2
│ │ │          │  轮胎服务             │ │
│ │ │ 缩略图   │  1号测试店铺            │ │  合并摘要，非四行表
│ │ │ 预览     │  小鹏 X9 · 12 张       │ │
│ │ │ aspect   │  [已完工][审核中]      │ │
│ │ │ Fill     │  更新于 06-08          │ │
│ │ └──────────┘  ▓▓▓▓▓░ 迷你进度       │ │
│ └────────────────────────────────────┘ │
│ ┌────────────────────────────────────┐ │  第二条（结构相同）
│ │ ┌──────────┐           分享  授权  │ │
│ │ │ 缩略图   │  …                      │ │
│ └────────────────────────────────────┘ │
│        … 继续滚动 …                      │
└──────────────────────────────────────────┘

点击卡片主体（除右上按钮）→ 详情翻页
点击分享 / 授权 → 列表内直接操作，不强制进详情
```

### 2.2 尺寸与密度（相对现 `AlbumCard`）

| 项 | 现卡（约） | 目标 |
| --- | --- | --- |
| 布局 | 纯文字，无图 | **左图右文** 横向 |
| 缩略图 | 无 / 极小 | `--size-album-list-thumb`（A-07，建议 **160～200rpx** 宽，与卡同高） |
| 卡片高度 | 偏扁 | **约 200～240rpx** 内容区 + 卡片 padding |
| 一屏条数 | 较多 | **2～4 条**（随机型）；禁止单条占满视口 |
| 相框感 | 无 | 缩略图可选 **轻相框角饰**（小面积，与详情相框 token 同源） |

### 2.3 卡片右上角 · 分享 + 授权/公示（合并）

| 控件 | 显示条件 | 文案/图标 | 行为 |
| --- | --- | --- | --- |
| 分享 | `canOwnerShare` 或已公开可分享 | 线型「分享」图标 | 打开 `ShareSheet`（albumId 上下文） |
| 授权/公示 | 见下表 | **单一按钮**，与状态合并 | 见下表 |

**授权/公示按钮（合并公示状态与操作）**

| `publicCaseStatus` | 按钮文案（示例） | 行为 |
| --- | --- | --- |
| `private` / 可授权 | 「授权公示」 | 打开授权流程（BottomSheet 勾选+提交，或跳授权页） |
| `pending_review` | 「审核中」 | 弱态；可点开说明「公开申请审核中」 |
| `public_approved` | 「已公开」 | 可点开「查看公示页」或弱化仅 Tag |
| `user_rejected` | 「授权公示」 | 同可授权 |
| 未完工 / 无图 | **不显示** | — |

图标+短文案同排；Tag 行只读状态，右上角为可操作入口（语义不重复堆砌）。

### 2.4 升级前后对比

| 维度 | 现 `AlbumCard` | 升级后（仍叫 `AlbumCard` / `ui-album-card`） |
| --- | --- | --- |
| 形态 | 扁文字卡 | **加高横向卡**，列表可滚动多条 |
| 图片 | 无预览 | **左侧缩略图**（首节点首图 / `coverUrl`） |
| 信息 | 分散多行 caption | 服务名突出 + 门店/车辆/张数合并 |
| 操作 | 商家端有分享 | 用户端右上 **分享 + 授权/公示** |
| 进度 | 无 | 可选六段迷你进度条 |

空态（私密）不变：暂无服务相册 · 扫码提示 · H5 公开案例链。

### 2.5 命名说明

- 产品口语「相册封面」= **卡片上的缩略图预览区**，不是全屏封面页。
- 实现上 **升级** `components/album-card`，**不新增**占满全屏的 `AlbumCoverCard` 组件类型。

## 3. 相册详情 · 全屏相框翻页（`pages/album/detail`）

> 主阅读 = **全屏相框 + 翻页**；**无**置顶摘要、**无**框内分享、**无**底栏授权按钮。  
> 组件：`ToolImmersiveNav` + `AlbumFrameViewer` + `AlbumBottomToolbar`（仅节点 Tab）+ `AlbumPageFooter` + `AlbumInfoSheet`。

### 3.1 页面态

| 态 | 布局 |
| --- | --- |
| `loading` | 沉浸导航 + 相框 Skeleton |
| `normal` | 翻页（含尾页 §3.6）+ 底栏 Tab + 页脚 |
| `empty` | 无过程图：Empty + 页脚（联系门店+合规） |
| `error` | Empty + 重试；无分享入口 |

### 3.2 主态线框

### 3.3 相框（`AlbumPhotoFrame`）

与 A-06 一致：四边等宽、角饰 L 形、立体阴影 `--shadow-album-frame`；艺术字仅框内短标签（≤4 字）。

### 3.4 图片页 · 框内区域

#### 翻页

- 横向 swiper；无露边、无大箭头；禁止 `previewImage`。
- 页序列：`[图1…图N]` + **尾页**（§3.6）。

#### 框内顶栏

| 元素 | 行为 |
| --- | --- |
| 页码 | `{index} / {total}` |
| 信息 | 打开 `AlbumInfoSheet`（§3.7） |

**无分享按钮**（分享在列表封面右上角 + 尾页）。

#### 框内底文案条（仅图片页）

| 字段 | 规则 |
| --- | --- |
| 节点名 | 当前页节点 `title` |
| 时间 | 节点更新时间 |
| 节点说明摘要 | **最多 1 行**省略；全文见信息抽屉 |

**无**「对此节点有疑问」链接（反馈移至尾页 §3.6）。

### 3.5 底栏 · 节点 Tab（`AlbumBottomToolbar`）

| 内容 | 说明 |
| --- | --- |
| 节点 Tab 横滑 | 当前节点高亮；点击跳到该节点第一张 |
| **已取消** | ~~授权公示按钮~~（A-06b） |
| **已取消** | ~~联系门店~~（移至 §3.8 页脚） |

### 3.6 尾页 · 空白结束页（`AlbumEndPage`）

翻到最后一张图后再滑一页，进入 **虚拟尾页**（无照片，暖底 + 轻相框内留白）。

```text
┌────────────────────────────────────┐
│                                    │
│         （无大图，档案纸底）        │
│                                    │
│    ┌──────────┐  按状态显示        │
│    │ 授权公示 │  或「审核中」说明   │
│    └──────────┘                    │
│    ┌──────────┐                    │
│    │   分享   │                    │
│    └──────────┘                    │
│    ┌──────────┐                    │
│    │   反馈   │  → 相册反馈页      │
│    └──────────┘                    │
│                                    │
└────────────────────────────────────┘
```

| 按钮 | 显示 | 行为 |
| --- | --- | --- |
| 授权公示 | `shouldShowAuth` | 同列表封面授权流程 |
| 分享 | 有可分享能力 | `ShareSheet` |
| 反馈 | 始终 | `album/feedback` |

尾页 **不显示** 框内底文案条；页码显示为最后一页（如 `13 / 13`）。

### 3.7 信息抽屉（`AlbumInfoSheet`）· 极简

> **仅**隐私 + 门店说明；**非**大段 unstructured 文字墙。

半屏 BottomSheet（约 **55–65%** 屏高）；标题「说明与隐私」。

```text
┌──────────────────────────────────────────┐
│  说明与隐私                        ✕     │
├──────────────────────────────────────────┤
│  [PrivacyBanner scene=album]              │
│  ─────────────────────────────────────   │
│  门店说明                                │
│  ┌────┬─────────────────────────────┐   │
│  │缩略│ 接车记录                      │   │  按节点分组
│  │图 │ 门店对该节点/图片的说明全文…  │   │
│  └────┴─────────────────────────────┘   │
│  ┌────┬─────────────────────────────┐   │
│  │缩略│ 施工过程 · 第2张            │   │  同节点多图可并组
│  │图 │ 说明全文…                    │   │
│  └────┴─────────────────────────────┘   │
│  … 可纵向滚动 …                          │
└──────────────────────────────────────────┘
```

| 区块 | 内容 |
| --- | --- |
| PrivacyBanner | 相册隐私短提示 |
| 门店说明列表 | 有 `note` 的节点：缩略图（该节点首图或当前图）+ 节点名 + **说明全文**（不截断） |
| 门店总说明 | 若有 `storeNote`，置顶一段「门店总说明」 |

**不在抽屉内**：摘要表、Tag、授权表单、分享、反馈、ComplianceNotice。

数据：Phase 1 节点级 `note`；无单图 `caption` 时以节点说明代表该组图片。

### 3.8 页底留白栏（`AlbumPageFooter`）

固定在 **页面最底部**（节点 Tab 下方），与相框主画布分层。

```text
┌──────────────────────────────────────────┐
│  [联系门店]   secondary / 全宽或左对齐      │
│  ─────────────────────────────────────   │
│  ComplianceNotice（HOME_PLATFORM_IDENTITY）│
└──────────────────────────────────────────┘
```

详情页 **ComplianceNotice 唯一位**在此；我的页等其他工具页仍各自页脚规则不变。

### 3.9 交互清单

| 操作 | 结果 |
| --- | --- |
| 左右滑 | 图片页 / 尾页 |
| 节点 Tab | 跳到节点首张（尾页时 Tab 仍可用） |
| 信息 | 打开极简抽屉 |
| 尾页 · 授权 | 授权流程 |
| 尾页 · 分享 | ShareSheet |
| 尾页 · 反馈 | 反馈页 |
| 页脚 · 联系门店 | 拨号 |
| 列表封面 · 分享/授权 | 不进入详情即可操作 |

### 3.10 禁止项

- 详情置顶摘要卡、抽屉内摘要表、底栏授权按钮  
- 框内顶栏分享、主屏 PrivacyBanner  
- 促销模块、时间线网格主视图

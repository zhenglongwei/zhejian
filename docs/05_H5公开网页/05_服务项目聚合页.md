## **2. 页面定位**

服务项目聚合页用于承接用户对具体维修保养项目的搜索需求。

示例：

```
刹车片更换案例
电瓶更换案例
小保养多少钱
钣喷修复案例
汽车空调维修

```

页面主要承担：

1. 说明服务项目是什么；
2. 说明什么情况需要做；
3. 展示维修流程；
4. 说明价格影响因素；
5. 聚合真实案例；
6. 推荐可做该项目的门店；
7. 引导用户预约服务。

## **5.2 项目说明**

说明该服务项目的基本含义。

示例：

```
刹车片更换是车辆制动系统常见保养项目之一。当刹车片磨损接近更换标准，或出现刹车异响、制动距离变长等情况时，建议到店检查刹车片、刹车盘和卡钳状态后确认是否需要更换。

```

要求：

1. 内容准确；
2. 不夸大；
3. 不承诺线上诊断；
4. 引导到店检测。

---

## **5.3 什么情况需要做**

以列表形式展示用户判断场景。

示例：

```
以下情况建议检查刹车系统：

1. 刹车时出现异响；
2. 刹车片磨损接近警戒线；
3. 仪表提示刹车片磨损；
4. 刹车脚感明显变化；
5. 制动距离变长；
6. 更换轮胎或保养时发现刹车片偏薄。

```

不同服务项目应配置不同内容。

---

## **5.4 维修流程**

展示标准服务流程。

刹车片更换示例：

```
1. 到店检查车辆和刹车系统；
2. 拆卸轮胎并检查刹车片厚度；
3. 检查刹车盘和卡钳状态；
4. 更换适配型号刹车片；
5. 复位安装并进行制动检查；
6. 试车确认刹车脚感和异响情况。

```

钣喷修复示例：

```
1. 检查损伤范围；
2. 评估是否涉及钣金修复；
3. 确认喷漆部位和修复方案；
4. 进行钣金整形和表面处理；
5. 喷涂、烘烤和抛光；
6. 完工检查。

```

## **5.5 参考价格/价格区间**

根据服务类型展示。

一口价服务：

```
参考价格：¥299 起

```

区间服务：

```
参考区间：¥300-¥1200

```

复杂维修：

```
需到店检测后确认维修方案和费用

```

事故车：

```
事故车维修无法仅凭线上图片准确报价，需到店检测或拆检后确认方案和费用。

```

价格说明：

```
页面价格为参考范围，实际费用会因车型、配件品牌、损伤程度、维修方案和门店检测结果不同而变化。

```

---

## **5.6 价格影响因素**

字段建议：


| 因素     | 说明              |
| ------ | --------------- |
| 车型品牌   | 不同车型配件和工时不同     |
| 配件品牌   | 原厂、品牌件、不同规格价格不同 |
| 损伤程度   | 钣喷、事故车尤其明显      |
| 工时复杂度  | 拆装、匹配、编程等影响工时   |
| 门店检测结果 | 最终以门店检测为准       |
| 城市差异   | 不同城市服务成本不同      |


---

## **5.7 真实案例**

展示该服务项目下的公开案例。

筛选条件：

1. 服务项目匹配；
2. 审核通过；
3. 已脱敏；
4. 允许公开；
5. 优先展示同城市案例；
6. 平台订单案例优先。

案例卡片字段：


| 字段   | 说明        |
| ---- | --------- |
| 主图   | 脱敏图       |

| 城市   | 展示        |
| 门店   | 展示        |
| 价格区间 | 可选        |
| 来源标签 | 平台订单/历史案例 |

## **5.8 推荐门店**

展示可提供该服务项目的门店。

推荐逻辑：

1. 上架了该服务商品；
2. 有该项目真实案例；
3. 门店评分较高；
4. 透明度评分较高；
5. 同城市优先；
6. 营业状态正常。

## **5.10 FAQ**

示例：

```
Q：刹车片更换一般需要多久？
A：常规情况下约 1-2 小时，具体时间取决于车辆状况和门店排队情况。

Q：刹车片一定要前后一起换吗？
```

## **9. noindex 规则**

以下页面不得开放 index：

1. 服务项目未上架；
2. 服务项目无说明内容；
3. 无真实案例且内容质量不足；
4. 页面只有模板文案；
5. 服务内容存在合规风险；
6. 页面重复度过高。

## **11. P0 验收标准**

1. 服务项目页有唯一 URL；
2. 页面包含项目说明；
3. 页面包含什么情况需要做；
4. 页面包含维修流程；
5. 页面包含价格影响因素；
6. 页面包含真实案例；
7. 页面包含推荐门店；
8. 页面包含 FAQ；
9. 页面支持打开小程序；
10. 页面具备 title、description、canonical、robots；
11. 页面不展示未审核或未脱敏内容；
12. 事故车相关服务明确不支持线上最终报价；
13. 页面访问和转化点击可埋点。

## **2. 页面定位**

车型聚合页用于承接车型相关搜索需求。

示例：

```
本田思域维修保养案例
宝马3系保养案例
奥迪A4L刹车维修案例
特斯拉Model 3保养维修

```

该页面主要承担：

1. 聚合车型维修保养信息；
2. 展示该车型常见保养项目；
3. 展示该车型常见故障；
4. 展示真实维修案例；
5. 展示价格参考；
6. 推荐擅长该车型的门店；
7. 引导用户预约相关服务。

---

## **3. 页面生成条件**

车型聚合页不建议批量生成空页面。

建议满足以下条件之一再生成：

1. 该车型已有不少于 3 个公开案例；
2. 该车型在平台有明确服务内容；
3. 该车型搜索需求较高；
4. 平台运营手动配置；
5. 城市 + 车型 + 服务组合有内容支撑。

不满足质量要求时：

- 不生成页面；
- 或生成但 noindex；
- 或合并到品牌级页面。

```

## **6.1 车型标题**

格式：

```
{车型}维修保养案例与价格参考

```

示例：

```
宝马3系维修保养案例与价格参考

如果是车型 + 服务：

```
宝马3系刹车片更换案例与价格参考

```

---

## **6.2 车型简介**

内容包括：

1. 车型品牌；
2. 车系；
3. 常见能源类型；
4. 常见年份区间；
5. 维修保养特点；
6. 平台案例数量，若有。

示例：

```
宝马3系是常见豪华品牌轿车，日常维修保养需求主要集中在小保养、刹车系统、轮胎、电瓶和底盘异响检查等项目。不同年款和配置的配件规格可能不同，实际维修费用需以门店检测和配件选择为准。

```

注意：

1. 不做品牌官方授权暗示；
2. 不写无法验证的通病结论；
3. 常见故障需谨慎表述为“常见关注点”。

---

## **6.3 常见保养项目**

展示与车型相关的服务项目。

字段：


| 字段   | 说明           |
| ---- | ------------ |
| 项目名称 | 如小保养、刹车片更换   |
| 服务说明 | 简短描述         |
| 参考价格 | 可选           |
| 案例数量 | 可选           |
| 入口   | 服务页/案例列表/小程序 |


示例项目：

```
小保养
刹车片更换
电瓶更换
轮胎更换
空调清洗
变速箱油更换

```

---

## **6.4 常见故障**

展示该车型用户常咨询的问题。

字段：


| 字段   | 说明          |
| ---- | ----------- |
| 故障名称 | 如刹车异响       |
| 可能原因 | 简要说明        |
| 建议检查 | 如刹车片、刹车盘、卡钳 |
| 相关案例 | 可点击         |


文案要求：

1. 使用“可能”“建议检查”等表达；
2. 不做线上诊断结论；
3. 不承诺一定原因；
4. 引导到店检测。

示例：

```
刹车异响可能与刹车片磨损、刹车盘表面状态、卡钳回位等因素有关，建议到店检查后确认维修方案。

```

---

## **6.5 真实维修案例**

展示该车型相关案例。

筛选逻辑：

1. 同车型优先；
2. 同品牌次之；
3. 同服务项目优先；
4. 同城市优先；
5. 公开、审核、脱敏案例。

案例卡片字段：


| 字段   | 说明        |
| ---- | --------- |
| 主图   | 脱敏图       |
| 标题   | 车型 + 服务项目 |
| 服务项目 | 展示        |
| 城市   | 展示        |
| 门店   | 展示        |
| 价格区间 | 可选        |
| 来源标签 | 平台订单/历史案例 |

## **6.7 推荐门店**

展示擅长该车型或相关服务的门店。

推荐逻辑：

1. 有该车型案例；
2. 有该品牌案例；
3. 有相关服务商品；
4. 门店评分高；
5. 透明度评分高；
6. 同城市优先。

门店卡片字段：


| 字段    | 说明    |
| ----- | ----- |
| 门店名称  | 展示    |
| 城市/区域 | 展示    |
| 擅长项目  | 展示    |
| 案例数量  | 展示    |
| 评分    | 可选    |
| 预约按钮  | 打开小程序 |


---

## **6.8 FAQ**

示例：

```
Q：宝马3系小保养一般多少钱？
A：费用会因机油品牌、用量、滤芯型号和门店工时不同而变化，建议查看平台服务页或到店确认。

Q：车型页面的价格是最终报价吗？
A：不是。页面价格为参考范围，实际费用需以门店检测和具体维修方案为准。

Q：可以直接预约擅长该车型的门店吗？
A：可以，点击页面中的预约入口可打开小程序查看门店和服务。

```

## **10. noindex 规则**

以下情况应 noindex：

1. 车型案例数量不足；
2. 页面主要内容为空；
3. 只有模板文案；
4. 没有真实案例；
5. 页面重复度过高；
6. 车型过于稀有，存在识别风险；
7. 内容未审核。

## **12. P1 验收标准**

1. 页面有唯一 URL；
2. 页面包含车型简介；
3. 页面包含常见保养项目；
4. 页面包含常见故障；
5. 页面展示真实维修案例；
6. 页面展示价格参考；
7. 页面展示推荐门店；
8. 页面包含 FAQ；
9. 页面具备 title、description、canonical、robots；
10. 无真实内容时不得开放 index；
11. 不暴露任何用户隐私；
12. 页面支持打开小程序。

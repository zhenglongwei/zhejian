## **2. 页面定位**

门店公开主页是商家在 H5 内容站的公开展示页。

主要承担：

1. 展示门店基础信息；
2. 展示门店认证和资质；
3. 展示擅长项目与车型；
4. 展示可预约服务商品；
5. 展示真实公开案例；
6. 展示透明度指标（案例数、相册完整率等）；
7. 展示透明度指标；
8. 引导用户进入小程序预约。

---

## **3. 页面访问条件**

门店页可展示的前提：

1. 门店已通过平台审核；
2. 门店状态为正常营业或暂停服务但允许展示；
3. 门店资料达到最低完整度；
4. 门店无严重违规；
5. 门店允许公开展示。

不可展示场景：


| 场景     | 处理                |
| ------ | ----------------- |
| 门店未审核  | 404 或 noindex     |
| 门店关闭   | 展示暂停服务或 404，视运营策略 |
| 门店严重违规 | 下线                |
| 门店资料不足 | noindex 或不生成页面    |
| 商家注销   | 页面下线              |

## **6.2 联系方式展示规则**

MVP 建议谨慎展示联系方式：

1. 微信内 H5 优先引导打开小程序联系；
2. 微信外可展示平台客服电话或二维码；
3. 门店电话是否展示由平台配置；
4. 不建议默认展示商家私人手机号；
5. 电话点击需埋点。

按钮：

```
打开小程序联系门店
预约该门店
查看服务

```

---

## **6.3 认证信息**

展示门店已通过的平台认证。

常见标签：

1. 平台审核通过；
2. 营业执照已认证；
3. 门店实景已认证；
4. 维修资质已认证；
5. 技师信息已完善；
6. 价格信息已完善；
7. 维修案例已公开。

字段：


| 字段   | 说明      |
| ---- | ------- |
| 认证名称 | 如营业执照认证 |
| 认证状态 | 已认证/未认证 |
| 认证说明 | 简要说明    |
| 更新时间 | 可选      |


不得展示未核实资质，不得伪造官方授权。

---

## **6.4 透明度指标**

透明度指标用于表达门店在平台上的可信展示程度。

建议展示：


| 指标      | 说明         |
| ------- | ---------- |
| 透明度评分   | 满分 100；无公开案例时不虚报 |
| 公开案例数   | 审核通过案例     |
| 过程资料齐全度 | **已完工**相册核心节点齐全比例（聚合，不链私密/进行中相册） |
| 服务信息完整度 | 服务商品资料完整情况 |
| 内容新鲜度   | 最近公开案例 / 资料核实时间（卷十一） |
| 资质核验    | 含有效期是否在有效期内 |


展示文案示例：

```
该门店已公开 18 个维修案例，近期过程资料齐全度约 86%，最近公开案例于 2026-07-10。

```

如数据不足：

```
该门店正在完善辙见资料。

```

**咨询响应**（线索已联系比例）：仅商家后台，**不对 H5 公开页展示**。

**硬规则**：进行中相册不得出现在门店公开面；可公示仅限已完工且用户授权（或商家历史案例）并审核通过的内容。

---

## **6.4.1 营业态与新鲜度（卷十一）**

| 字段 | 说明 |
| ---- | ---- |
| 门店状态 | `open`/`closed`/`holiday`/`suspended`/`offline`；须按营业时间+临时休息实时计算，禁止把「已上线」直接当「营业中」 |
| 临时休息 | 商家维护的日期区间说明 |
| 新鲜度摘要 | 最近公开案例时间、资料最近核实时间 |

## **6.6 擅长车型**

展示门店擅长的品牌或车系。

示例：

```
宝马3系
奥迪A4L
本田思域
奔驰C级
特斯拉Model 3

```

展示规则：

1. 来源于门店配置与公开案例统计（**不得**用已删除的订单数据）；
2. 不得虚假标注；
3. 如果案例不足，可展示品牌级别，不展示具体车系；
4. 点击可进入车型聚合页或案例列表。

### **6.6.1 技师 / 设备 / 暂不承接（卷十一）**

| 区块 | 规则 |
| ---- | ---- |
| 技师公示卡 | 须平台审核通过后展示 |
| 设备/场标签 | 须平台审核通过后展示 |
| 暂不承接 | 商家维护，即时展示；空则隐藏 |
| 品牌授权 | 审核通过且在有效期内才展示 |

---

## **6.7 服务商品**

展示门店可预约服务。

服务卡片字段：


| 字段   | 说明           |
| ---- | ------------ |
| 服务名称 | 如刹车片更换       |
| 服务类型 | 一口价/区间价/到店检测 |
| 参考价格 | 如 ¥299 起     |
| 适用车型 | 可选           |
| 服务说明 | 简要           |
| 预约按钮 | 打开小程序        |


价格展示规则：

1. 一口价服务展示明确价格；
2. 区间价服务展示参考区间；
3. 事故车服务展示“到店检测后确认”；
4. 下架服务不展示预约按钮。

---

## **6.8 真实案例**

展示门店已公开案例。

排序建议：

1. 最新案例；
2. 高浏览案例；
3. 与当前主服务相关案例；
4. 用户授权案例优先。

案例卡片字段：


| 字段   | 说明        |
| ---- | --------- |
| 主图   | 脱敏图       |
| 标题   | 车型 + 服务项目 |
| 服务项目 | 展示        |
| 价格区间 | 可选        |
| 来源标签 | 用户授权/商家历史 |
| 发布时间 | 可选        |


---

## **6.9 透明度指标**（V2.0 替代交易评价）

展示门店可验证的透明度数据，**不展示**订单交易评价。详细口径见 **§6.4**（卷十一加固）。

| 字段 | 说明 |
| --- | --- |
| 公开案例数 | 已审核脱敏案例 |
| 过程资料齐全度 | **已完工**相册聚合比例，不链进行中/私密相册 |
| 内容新鲜度 | 最近公开案例 / 资料核实 |
| 擅长项目标签 | 来自门店资料 |
| 资质认证 | 平台审核通过（含有效期） |

规则：

1. 数据须来自平台真实统计，不得伪造；
2. MVP 不做星级评价墙、好评率等交易评价展示；
3. 若暂无案例，展示空状态而非虚构数据；
4. **咨询响应不对 H5 展示**；进行中相册不得出现在公开面。

## **11. 异常处理**


| 场景     | 处理                |
| ------ | ----------------- |
| 门店不存在  | 404               |
| 门店审核中  | 404/noindex       |
| 门店暂停营业 | 展示暂停服务，不显示预约或弱化预约 |
| 门店严重违规 | 下线                |
| 无公开案例  | 展示空状态，不影响门店页      |
| 无服务商品  | 展示“暂无可预约服务”       |
| 电话未公开  | 不展示电话             |

## **12. P0 验收标准**

1. 门店主页有唯一 URL；
2. 页面包含门店基础信息；
3. 页面包含认证信息；
4. 页面包含擅长项目；
5. 页面包含服务商品；
6. 页面包含真实案例模块；
7. 页面包含透明度指标模块；
8. 页面包含透明度指标；
9. 页面包含预约入口；
10. 页面支持打开小程序；
11. 页面具备 title、description、canonical、robots；
12. 门店关闭或违规时可下线；
13. 不展示未审核图片和隐私信息；
14. 页面访问与预约点击可埋点。

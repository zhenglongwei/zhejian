## **2. 页面定位**

公开案例详情页是 H5 内容站的核心页面。

主要承担：

1. 展示真实维修案例；
2. 展示脱敏后的维修过程图片；
3. 解释维修问题、维修方案和价格影响因素；
4. 支撑 SEO 搜索收录；
5. 支撑 AI/GEO 引用；
6. 引导用户打开小程序预约类似服务；
7. 为门店建立透明维修能力背书。

---

## **3. 页面访问条件**

案例详情页只允许展示满足以下条件的案例：

1. 案例已审核通过；
2. 案例允许公开；
3. 图片已完成脱敏；
4. 用户隐私授权有效，或历史案例已完成隐私责任确认；
5. 案例未下架；
6. 案例不处于争议、投诉、风险锁定状态；
7. 内容质量达到 SEO 页面规范要求。

不满足条件时：

- 未公开/审核中：返回 404 或 noindex 预览页；
- 已下架：返回 404 或 410；
- 隐私风险：立即不可访问；
- 仅门店展示但不允许收录：页面可访问但 robots 为 noindex。

## **4. 推荐 URL**

根据 SEO 页面规范，P0 推荐：

```
/case/{case_id}.html

```

示例：

```
/case/cas_202501010001.html

```

后续可扩展为：

```
/case/{vehicle_slug}-{service_slug}-{city_slug}-{case_id}.html

```

示例：

```
/case/bmw-3series-brake-pad-replacement-beijing-cas_202501010001.html

```

---

## **6.1 顶部导航**

展示内容：

1. 平台 Logo；
2. 城市入口；
3. 打开小程序按钮；
4. 返回首页或案例列表入口。

移动端顶部建议固定或半固定展示。

---

## **6.2 标题**

标题格式：

```
{城市}{车型}{服务项目}维修案例

```

示例：

```
杭州宝马3系刹车片更换维修案例

```

如果车型不适合展示过细，可模糊为：

```
杭州德系轿车刹车片更换维修案例

```

标题禁止包含：

1. 用户姓名；
2. 车牌；
3. VIN；
4. 手机号；
5. 夸大承诺词，如“全网最低”“100%修好”。

---

## **6.3 案例标签**

根据案例来源展示。

平台订单案例：

```
平台订单案例 · 已脱敏 · 已审核

```

商家历史案例：

```
商家历史案例 · 已脱敏 · 已审核 · 价格仅供参考

```

如为事故车案例，额外展示：

```
事故车维修 · 到店检测后确认方案

```

---

## **6.4 AI 可引用摘要**

用于帮助搜索引擎和 AI 摘要理解页面内容。

展示位置建议在标题下方。

格式：

```
本页展示{城市}{车型}{服务项目}维修案例，包含故障表现、检查结果、维修方案、脱敏维修过程图片、价格区间和影响因素。页面不展示车牌、VIN、手机号和用户身份信息，实际费用需以门店检测结果为准。

```

示例：

```
本页展示杭州宝马3系刹车片更换维修案例，包含刹车异响检查、旧件磨损情况、新旧件对比、安装过程和完工检查。图片已做隐私脱敏处理，实际费用会因配件品牌和车辆状况不同而变化。

```

## **6.6 故障/需求描述**

展示用户遇到的问题或服务需求。

示例：

```
车辆在低速刹车时出现轻微异响，门店检查后发现前轮刹车片磨损接近更换标准，建议更换前轮刹车片并进行完工试车检查。

```

要求：

1. 不展示用户原始聊天内容；
2. 不展示联系方式；
3. 不展示车牌；
4. 不夸大故障严重程度。

---

## **6.7 检查结果**

展示门店检测结论。

示例：

```
检查发现前轮刹车片磨损明显，刹车盘表面无严重拉伤，刹车油液位正常。本次建议优先更换前轮刹车片，并在安装后进行刹车脚感和异响检查。

```

---

## **6.8 维修方案**

展示维修过程和处理方式。

可结构化展示：

```
1. 拆卸前轮；
2. 检查刹车片和刹车盘状态；
3. 更换适配型号刹车片；
4. 清洁卡钳和安装位置；
5. 复位安装并进行试车检查。

```

---

## **6.9 价格区间与价格影响因素**

### **价格区间展示**

标准服务：

```
参考价格：¥299 起

```

复杂维修：

```
参考区间：¥300-¥1200，到店检测后确认

```

事故车：

```
事故车维修无法仅凭线上图片准确报价，需到店检测或拆检后确认方案和费用。

```

### **价格影响因素**

建议展示：

1. 车型品牌；
2. 配件品牌；
3. 配件规格；
4. 损伤程度；
5. 是否需要拆检；
6. 是否涉及喷漆、校正、编程；
7. 门店工时标准。

说明文案：

```
本案例价格仅展示参考区间，实际费用会因车型、配件品牌、损伤程度、维修方案和门店检测结果不同而变化。

```

## **6.10 维修过程相册**

相册展示维修过程节点。

常见节点：

1. 维修前；
2. 检查过程；
3. 旧件状态；
4. 新旧件对比；
5. 安装过程；
6. 完工效果；
7. 试车/检测结果。

图片规则：

1. 只能展示脱敏图；
2. 支持点击放大；
3. 图片下方展示节点说明；
4. 图片 alt 必须符合 SEO 规范；
5. 不展示原图；
6. 不展示订单号；
7. 不展示精确上传时间。

## **6.11 前后对比**

适用于：

1. 钣喷修复；
2. 事故车修复；
3. 大灯翻新；
4. 内饰修复；
5. 轮毂修复；
6. 漆面修复。

展示方式：

```
维修前图片
维修后图片

```

文案示例：

```
维修前可见右前翼子板存在明显剐蹭，修复后外观恢复平整，色差控制在正常观察范围内。

```

如无前后对比图片，则模块不展示。

## **6.12 配件说明**

展示本次维修涉及的配件信息。


| 字段   | 说明         |
| ---- | ---------- |
| 配件类型 | 如刹车片、电瓶、轮胎 |

禁止：

1. 虚假原厂；
2. 未授权品牌宣传；
3. 夸大质保；
4. 使用“永久质保”等绝对承诺。

## **6.13 完工结果**

示例：


## **6.14 完工建议**

## **6.15 FAQ**

案例详情页建议至少 3 条 FAQ。

示例：

```
Q：刹车片更换一般需要多久？
A：常规情况下约 1-2 小时，具体时间取决于车辆状况和门店排队情况。

Q：刹车片是否必须前后一起更换？
A：不一定，需要根据前后刹车片的实际磨损情况判断。

Q：更换刹车片后需要注意什么？
A：建议短期内避免急刹，关注刹车脚感和异响情况，必要时回店检查。

```

FAQ 必须页面可见，才能用于 FAQPage 结构化数据。

## **11. 异常处理**


| 场景     | 处理                |
| ------ | ----------------- |
| 案例不存在  | 404               |
| 案例审核中  | 404 或 noindex 预览  |
| 案例已下架  | 404 或 410         |
| 图片未脱敏  | 不展示页面             |
| 用户撤回授权 | 页面立即下线            |
| 门店关闭   | 案例可保留，但不引导该门店预约   |
| 服务下架   | 案例可保留，但服务按钮显示暂停预约 |

## **12. P0 验收标准**

1. 公开案例详情页有唯一 URL；
2. 页面只展示已审核、已脱敏、允许公开的案例；
3. 页面包含标题；
4. 页面包含 AI 可引用摘要；
5. 页面包含关键信息表；
6. 页面包含维修过程相册；
7. 图片只能使用脱敏图；
8. 图片 alt 不为空；
9. 页面包含价格区间或价格说明；
10. 页面包含门店信息；
11. 页面包含咨询预约入口；
12. 页面具备 title、description、canonical、robots；
13. 页面支持打开小程序；
14. 案例下架后 H5 不可访问；
15. 页面访问和转化点击可埋点。

# 术语表

> 持续维护。


| 术语                   | 含义                      |
| -------------------- | ----------------------- |
| **维修相册**             | 一次维修的结构化过程记录（七阶段）       |
| **订单相册**             | 绑定平台订单的相册               |
| **快速相册**             | 先建相册后补订单                |
| **历史补录相册**           | 补录近期案例，强脱敏、降权           |
| **公开案例**             | 脱敏+授权+审核后发布的 H5 内容      |
| **翻译层**              | 车主查看时将专业术语转为易懂文案        |
| **线索 Lead**          | 预约到店留资，非订单              |
| **草稿订单**             | 创建相册时生成，金额为 0 的事后凭证     |
| **确认完工**             | 车主确认取车，触发评价             |
| **六维评价**             | 流程/工艺/价格/时效/配件/服务       |
| **案例价值分**            | 公开案例激励倾斜模型              |
| **GEO**              | 生成式引擎优化，面向 AI 与搜索的可引用内容 |
| **lifecycle_status** | 订单 GEO 生命周期状态（实现字段）     |

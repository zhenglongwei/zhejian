## 1. 运营目标（验证期 · 60～90 天）

验证一句话：

> 门店愿不愿意持续用现有服务相册、愿不愿意推动案例发布；公开案例能不能攒起来。接受度高再扩团队、加深确认/证据包与正式收费工程。

**验证期主指标**（对齐 `10` §3.1 过线表）：

| 指标 | 参考阈值 |
| --- | --- |
| 稳定合作店 | ≥8 家 |
| 供给 | 多数店每周 ≥2 完工相册 |
| 公开案例 | 合计 ≥100，或店均 ≥15 |
| 付费信号 | ≥3 家愿续费或表示离不开 |

**观察指标**：A/B 类到店提及、电话触点、GEO。  
**验证期不考核**：确认完成率、证据包次数（能力未上线）。  
**不以**「H5 留资爆表」「大模型推荐量」为过线条件。

对应开发任务：`BIZ-VAL-01～04`（见 `00_开发计划.md` §4.5.1）。

## 2. 套餐与样板政策（对外执行口径）

与产品文档第二轮一致，对外只讲：

| 套餐 | 价格 | 对外一句话 |
| --- | --- | --- |
| 体验版 | 0 元（月额度有限） | 试用过程留证与车主查看 |
| 专业版 | **480 元/年**（已定：低门槛盖成本 + 付费认真感） | 透明成交与合规留证专用工具 |

- **99 元档、980 获客/收录主推：一律下线**，话术与海报不得再出现。  
- **发布到公开网站 / 基础收录：不另收费**（须车主自己点发布 + 平台审核）。  
- 存量旧套餐：到期续费引导至专业版。

### 2.1 杭州创始样板店（前 10～15 家）· 验证期

| 项 | 内容 |
| --- | --- |
| 试用 | **工具宽松试用 90 天**（不强制先上线额度阻断工程） |
| 续费意向 | 验证期末收集「愿否续专业版」；正式半价政策在**过线后**套餐工程落地时执行 |
| 平台义务 | 拍照/录入规范、首批案例协助、**每周 1 次**短复盘 |
| 门店义务 | 每周 ≥2 个真实完工相册；推动「发布到公开网站」；回传到店（观察） |
| 退出 | 连续 2 周不达标 → 取消代运营类支持 |

**验证期对外**：公开不另收费；不承诺 AI 送客；确认/证据包未上线则不说「已具备完整合规阻断」。

### 2.2 线索与获客（样板期）

- **只记帐、不向门店结算**。  
- 明确告知：当前付的是**工具**；网上公开不另收费；获客效果先一起看数据。  
- 以后若单开线索/专题包，另签，不绑架专业版续费话术。

| 指标 | 目标 |

3. 平台参与报价、质保、售后  
4. 「不买专业版就不能上公开网站」（与公示免费冲突，禁止）  
5. 把高德/美团来电变多写成合同验收条款

## 4. 对外定位与话术

### 4.1 定位

**透明成交与合规留证的专用工具。**  
现有开单/收银软件继续用；辙见补的是：过程说得清、车主认得清、出事拿得出证据；愿意公开时晒得干净。

GEO/网上案例是**长期加分**，不是现在收的钱。

### 4.2 现实路径（先对齐老板预期）

客人常：弄懂怎么修 → 高德/美团看店 → 电话或到店。  
辙见帮的是到店前信任与到店后少扯皮，**不是**替代高德/美团下单。

### 4.3 招商开口（约 2 分钟）

**不要说**：  
「一年收录/获客版，AI 就会推荐你，客户会在我们网站留电话。」

**要说**：  
「你们现在的系统继续开单收款。辙见是专用工具：把维修过程留给车主看清，关键配件让车主确认留痕，出事能导出证据。公开案例车主愿意发就发，**不另收费**。专业版一年 **480 元**，门槛不高；样板店先免费用 90 天，好用再续。网上能不能多来客，我们一起看，但不拿这个当保证。」

### 4.4 物料短句

- 开单用你现有系统；确认与留证用辙见。  
- 过程看得见，举证拿得出。  
- 公开案例不另收费；专业版买的是店里每天用的工具。

## 6. 90 天作战计划

### 第 0～2 周：定供给与预期

- 签 8～12 家；指定店内责任人  
- 聚焦易拍项目；协助首批约 5 个可公开案例  
- **开场就讲清**：付费点是工具；公示免费；不承诺 AI 送客  
- 禁止：扩城、99/980 旧价话术  

### 第 3～6 周：提活跃与发布率

- 交车时邀请车主查看相册；愿公开则走「发布到公开网站」  
- 发布率低：改话术（帮后来车主避坑），不要改成「不发就不能用工具」  
- 公众号可发案例摘要强绑门店电话（内容分发，非套餐售卖）  

### 第 7～10 周：验证工具付费

- 每店一页小报：相册活跃、公开数、（若有）确认/证据包使用、续费意向  
- 约谈专业版续费；记录嫌贵 / 嫌麻烦 / 觉得和 ERP 重复  
- 若反馈「和开单软件重复」→ 产品侧加强 C1/C2 话术与轻量路径，勿硬推 GEO  

### 第 11～12 周：定主轴

| 若结果是… | 则下一步… |
| --- | --- |
| 活跃高、愿续费 | 锁定工具专业版为主商业；公示维持免费 |
| 活跃低、嫌重 | 减负录入；暂缓确认强阻断范围 |
| 活跃有、不愿付 | 检查体验版是否过宽；或价锚是否过高 |
| 只有「要流量才续」 | 勿立刻改回收录费；先诚实降预期或单开观察型分发试点 |

## 7. 对外招商一页纸（可复制）

### 辙见是什么

透明成交与合规留证的专用工具。  
跟现有开单、收银、进销存软件**一起用**，不替换。  
帮你：过程说清楚、车主认得清、出事拿得出证；车主愿意时，真实案例可发到公开网站（**不另收费**）。

### 体验版｜0 元

- 有限额度的服务相册与车主查看  
- 私域分享  
- 发布到公开网站（须车主操作 + 审核）**免费**  

### 专业版｜480 元/年

- 更高额度 / 不限量使用  
- 完整留证与（能力上线后）确认、交车报告、证据包等专业能力  
- 公开同样不另收费  
- 工具使用统计  

**杭州样板店**：专业版试用 90 天；协助规范与复盘；结束后首年半价。需稳定上传真实案例并回传到店情况。

### 保证 / 不保证

| 做得到 | 不承诺 |
| --- | --- |
| 过程留证与车主查看工具 | 大模型一定推荐 |
| 合规向的确认与举证能力（按产品节奏上线） | 搜索排名与成交额 |
| 公开案例基础展示不另收费 | 替代高德/美团下单 |
| 付费不买综合排名 | 保证站内留资条数 |

### 三步开始

1. 开通工作台（现有 ERP 照常使用）  
2. 按规范做首批服务相册  
3. 车主查看；愿公开则发布到公开网站

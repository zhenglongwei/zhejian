**说明**

- 「底盘异响 / 胶套」目前是**教练扩展包** `chassis_noise`，不是独立相册模板；不纳入模板硬清单。
- 平台服务目录（可上架项）少于模板数：保养、刹车、钣喷、电瓶、事故、自定义；轮胎/空调/大保养主要靠**创建相册时关键词 → 模板**。

**教练匹配原则（改规则时遵守）**

1. 优先 `templateId` 精确匹配规则包；  
2. 其次服务名关键词；  
3. 未命中 → `default` 中性包；  
4. **禁止**用某一类维修的专属示例充当「通用」备注样例。

## 2. 映射方法（各类模板通用）

每类服务、每个阶段填四列：

| 列 | 含义 |
| --- | --- |
| 车主关心 | 到店前/验收时最怕什么、最想确认什么 |
| AI / GEO 可引用点 | 公开案例、专题、FAQ 能沉淀的事实（无金额、无绝对化承诺） |
| 建议拍 | `shoot_prefer`；标注公域价值 / 仅留档 |
| 备注怎么写 | 句式骨架 + **本类型专用示例**（禁止串类型） |

叠加全局：`shoot_avoid`（隐私）只写中性项；公域藏价；完工 LLM 只吃文字。

GEO 三维标签（可写入 `geo_angle`）：

| 编码 | 取向 |
| --- | --- |
| `avoid_pitfall` | 现象 + 判断依据 + 方案对比（反对过度维修叙事） |
| `standard_5s` | 防护、工序、环境规范（打消「糊弄施工」顾虑） |
| `liability` | 拆件确认、材料说明、质保边界 |

## 3. 钣喷样板（`body_paint`）· 已验证

> 以下为已定稿样板，结构供其余模板复用。规则见 `album-coach-rules.js` · `body_paint`。

### 3.1 双关注点总表（跨阶段）

#### 车主关心

| 主题 | 典型问法 | 主要落点阶段 |
| --- | --- | --- |
| 色差 | 修完会不会花？金属漆/珍珠漆更怕什么？ | 检测、方案、施工、交付 |
| 该不该整面喷 | 小划痕一定要整面吗？补漆 vs 钣金喷漆？ | 检测、方案 |
| 工艺是否缩水 | 有没有打磨/中涂/烤漆？ | 施工、交付 |
| 拆不拆件 | 门把手饰条拆喷还是遮蔽喷？ | 方案、施工 |
| 材料 | 用什么漆？水性/油性？ | 配件、方案 |
| 工期与养护 | 多久能取？多久能洗车？ | 方案、交付 |
| 质保 | 起泡开裂管不管？ | 交付 |
| 隐私 | 整车照会不会被人认出车牌？ | 全阶段避忌 |

#### AI / GEO 关心

| 主题 | 希望案例里出现的「证据句」 | 主要落点 |
| --- | --- | --- |
| 部位 + 损伤类型 | 「右前门中度划痕，未见钣金变形」 | 接车/检测 |
| 工艺路径选择 | 「建议局部补漆，不整面喷」及判断依据 | 检测/方案 |
| 工序完整度 | 打磨 → 腻子/中涂 → 调色 → 喷涂 → 烤漆/抛光 | 施工 |
| 验收动作 | 自然光色差复查 | 交付 |

### 3.2 六阶段要点（摘要）

| 阶段 | 建议拍重点 | 备注示例（钣喷） |
| --- | --- | --- |
| stage_1 | 损伤远景，一处一图 | 「右前门、右后视镜壳划痕到店；外观未见新增磕碰。」 |
| stage_2 | 近景/多角 | 「右前门中度划痕，漆下未见钣金变形；建议局部补漆，不必整面喷。」 |
| stage_3 | 方案文字（报价单仅留档） | 「局部补漆不拆件；珍珠漆需调色过渡。」 |
| stage_4 | 漆/辅料包装标签 | 「本次使用色漆+清漆；包装批号已拍照，可按品牌官网核对。」 |
| stage_5 | 遮蔽、关键工序 | 「已打磨并完成调色比对；交车前自然光复查色差。」 |
| stage_6 | 完工近景、色差查 | 「自然光下过渡区复查；养护期内避免高压水枪。」 |

### 3.3 合规红线（钣喷）

- 不写公域金额；不写「100% 无色差」；不羞辱同行；公开避车牌/人脸/钥匙。

## 4. 刹车（`brake`）

### 4.1 双关注点

| 车主关心 | AI / GEO 点 |
| --- | --- |
| 是不是必须换？只换片还是连盘？异响/刹车软是片还是其他？ | 厚度实测读数；「建议换 / 可暂缓」边界；力矩与排气/磨合说明 |
| 配件真伪、是否原厂件 | 包装/编码凭证；不冒充平台鉴定 |
| 换完安不安全、要不要磨合 | 力矩打卡、路试/试车结论 |

### 4.2 六阶段映射

| 阶段 | 建议拍 | 备注示例 |
| --- | --- | --- |
| stage_1 | 到店相关：轮毂/制动器外观（勿强求拆轮） | 「到店诉求：刹车异响/行程变长；外观未见新增磕碰。」 |
| stage_2 | **片厚实测读数**；盘面状态近景 | 「外侧片剩余约 3mm；盘面未见明显沟槽；建议更换刹车片，盘可暂不换。」 |
| stage_3 | 方案文字（仅换片/片+盘） | 「建议更换前片；盘厚度与平整度可继续使用，无需连带更换。」 |
| stage_4 | 片/盘包装与编码特写 | 「前刹车片已开封核对规格；包装编码已拍照，可按品牌渠道核对。」 |
| stage_5 | **安装扭矩打卡**；卡钳回位/管路规范可见点 | 「按规范力矩紧固；磨合建议已当面告知（无绝对化承诺）。」 |
| stage_6 | 旧件对照、试车结论 | 「旧件已交还对照；短途试车异响消失、踏板行程正常。」 |

geo 侧重：`avoid_pitfall`（检测）+ `standard_5s`（施工）。

### 4.3 合规

- 不写「不换马上出事」恐吓；用实测厚度说话。  
- 不写公域金额；磨合说明勿绝对化。

## 5. 保养（`maintenance` / `major_maintenance`）

> **同规则包**：大保养与小保养共用 `maintenance` 包；差异主要在节点标题与项目多少，教练示例用「机油机滤等」可覆盖，大保可在备注中列出附加项目名。

### 5.1 双关注点

| 车主关心 | AI / GEO 点 |
| --- | --- |
| 做了哪些项目？机油标号对不对？ | 项目清单；防伪/液位；机舱清洁 |
| 有没有乱加项目、过度保养 | 「做了什么 / 建议暂缓什么」 |
| 现场是否规范、会不会弄脏内饰 | 防护垫、5S |

### 5.2 六阶段映射

| 阶段 | 建议拍 | 备注示例 |
| --- | --- | --- |
| stage_1 | 到店仪表里程/保养灯相关（勿拍清晰导航轨迹） | 「到店常规保养；里程与保养提示已核对。」 |
| stage_2 | 机油状态/滤芯外观等检查点（如有） | 「机油已到更换周期；空气滤外观尚可，建议下次再换。」 |
| stage_3 | 保养项目清单文字 | 「机油+机滤；空调滤按需；暂不更换火花塞。」 |
| stage_4 | 机油桶/滤芯包装防伪区 | 「机油规格与车型匹配；防伪与批次已拍照，可按品牌官网核对。」 |
| stage_5 | **防护垫与工具车**；加注/液位 | 「机油机滤已更换；机舱清洁复查；液位在标尺范围。」 |
| stage_6 | 保养后仪表/单据要点（金额勿入镜） | 「项目已完成；下次保养里程建议已告知。」 |

geo 侧重：`standard_5s`；方案阶段可带 `avoid_pitfall`。

### 5.3 合规

- 不写「必须用某进口油否则伤发动机」类绝对化。  
- 单据含金额、姓名时勿作公开展示素材。

## 6. 电瓶（`battery`）

### 6.1 双关注点

| 车主关心 | AI / GEO 点 |
| --- | --- |
| 打不着火是不是电瓶？要不要换？ | 电压/启动电流等检测结论；排除桩头/发电机 |
| 规格匹不匹配、有没有质保 | 规格标签、安装固定、旧件交接 |
| 编程/防盗匹配（如有） | 备注说明是否需要匹配，勿夸大 |

### 6.2 六阶段映射

| 阶段 | 建议拍 | 备注示例 |
| --- | --- | --- |
| stage_1 | 到店启动相关现象说明（可拍电瓶舱远景） | 「到店无法启动/启机无力；外观未见新增磕碰。」 |
| stage_2 | 电瓶检测仪读数；桩头状态 | 「负载测试偏低；桩头无严重腐蚀；建议更换电瓶，发电机待复查。」 |
| stage_3 | 更换方案与规格 | 「更换匹配规格电瓶；启停车型需确认规格，不做过度升级推销。」 |
| stage_4 | 新电瓶规格标签/包装 | 「规格与车型匹配；包装信息已拍照，可按品牌渠道核对。」 |
| stage_5 | 安装固定、桩头紧固 | 「已安装固定并紧固桩头；通电自检正常。」 |
| stage_6 | 启动验证；旧件交接 | 「启动正常；旧电瓶已交还/按环保要求回收说明已告知。」 |

geo 侧重：`avoid_pitfall`（检测）+ `liability`（交接/质保）。

### 6.3 合规

- 不写「打不着火一定是电瓶」；强调检测排除。  
- 不写公域金额。

## 7. 轮胎（`tire`）

### 7.1 双关注点

| 车主关心 | AI / GEO 点 |
| --- | --- |
| 要不要四条一起换？偏磨原因？ | 花纹深度、偏磨现象、是否建议四轮定位 |
| 动平衡做了没？气压对不对？ | 动平衡/气压记录可见点 |
| 品牌规格是否一致 | 胎侧规格特写 |

### 7.2 六阶段映射

| 阶段 | 建议拍 | 备注示例 |
| --- | --- | --- |
| stage_1 | 到店轮胎外观/亏气相关部位 | 「到店诉求：更换磨损轮胎；外观未见新增磕碰。」 |
| stage_2 | 花纹深度/偏磨近景 | 「花纹接近磨损标记；外侧偏磨，建议换胎后做四轮定位。」 |
| stage_3 | 更换条数与是否定位 | 「更换两前轮同规格轮胎；建议四轮定位，非强制捆绑销售。」 |
| stage_4 | 新胎胎侧规格/DOT 可读区 | 「规格与原车一致；胎侧信息已拍照留档。」 |
| stage_5 | 拆装过程规范点；动平衡 | 「已完成动平衡；气压按门店标准设定。」 |
| stage_6 | 装车后外观；路感说明 | 「装车完成；建议磨合期注意胎噪与路感变化。」 |

geo 侧重：`avoid_pitfall` + `standard_5s`。

### 7.3 合规

- 不恐吓「必须四条一起换」；用磨损与安全边界说明。  
- 少拍清晰车牌整车。

## 8. 空调（`ac`）

### 8.1 双关注点

| 车主关心 | AI / GEO 点 |
| --- | --- |
| 不制冷/异味/风量小分别啥原因？ | 压力/检漏/滤芯状态等检测点；排除项 |
| 是否必须洗箱、换压缩机？ | 轻量方案 vs 大修边界 |
| 冷媒是否合规加注 | 加注过程可见点（勿拍金额单） |

### 8.2 六阶段映射

| 阶段 | 建议拍 | 备注示例 |
| --- | --- | --- |
| stage_1 | 到店诉求相关（出风口/异味描述） | 「到店诉求：制冷变弱/有霉味；外观未见新增磕碰。」 |
| stage_2 | 压力表/检漏相关可读点；滤芯外观 | 「系统压力偏低；蒸发箱异味可疑；建议先换滤芯并检漏，暂不拆箱。」 |
| stage_3 | 维修路径 | 「检漏+补冷媒；滤芯更换；暂不清洗蒸发箱。」 |
| stage_4 | 滤芯/冷媒相关包装标签 | 「空调滤芯规格匹配；材料包装已拍照留档。」 |
| stage_5 | 加注/更换滤芯过程规范点 | 「已更换滤芯并按规范加注；通电试冷中。」 |
| stage_6 | 出风温度/异味复查 | 「出风明显变凉；异味减轻；建议定期更换滤芯。」 |

geo 侧重：`avoid_pitfall` + `standard_5s`。

### 8.3 合规

- 不写「一定要洗箱否则伤肺」类绝对化。  
- 冷媒操作符合门店规范表述即可，勿暗示违规排放。

## 9. 事故车（`accident`）· 弱公开

> 公域宜少、私域宜全。公开侧强调流程与责任边界，**不写金额、不线上估价**。

### 9.1 双关注点

| 车主关心 | AI / GEO 点 |
| --- | --- |
| 损伤范围、要不要拆检 | 外观+关键检结论；定损沟通节点（无金额） |
| 配件原厂/再制造边界 | 配件类型说明、凭证 |
| 保险与自费边界 | 责任/项目归属一句话（无价税数字） |
| 交车验收 | 与检测同序对照 |

### 9.2 六阶段映射

| 阶段 | 建议拍 | 备注示例 |
| --- | --- | --- |
| stage_1 | 外观多面、里程；事故概况文字 | 「追尾到店；右后部损伤；里程已记录。」 |
| stage_2 | 损伤近景；必要读数/拆检点 | 「外板变形，内结构待拆检确认；已排除明显漏液。」 |
| stage_3 | 维修方案要点（更换/钣金/喷漆组合） | 「右后翼子板建议更换；尾灯总成待定损确认后施工。」 |
| stage_4 | 新件包装与编码 | 「翼子板包装编码已核对；可按品牌渠道核对。」 |
| stage_5 | 关键拆装/校正/喷漆节点（避车牌） | 「已完成钣金校正与试装；喷漆工序按店内流程。」 |
| stage_6 | 完工对照；验收项 | 「外观对照检测顺序复查；交车说明与质保边界已告知。」 |

geo 侧重：`liability` 为主；检测可带 `avoid_pitfall`。

### 9.3 合规（加强）

- **禁止**公域任何成交价/定损金额数字。  
- **禁止**「线上即可确认最终价格」。  
- 案例仅作流程认知，不能替代本车检测。

## 1. 两层模型

| 层 | 存储 | 版本字段 | 可变 |
| --- | --- | --- | --- |
| **CaseSnapshot** | `public_cases.content_json.snapshot` | `content_json.snapshot.version`（与业务 `snapshotVersion` 对齐） | **否**（公示后） |
| **GeoEnrichment** | `public_cases.enrichment_json` | `enrichment_version` | **是** |
| **商家润色草稿** | `albums.content_optimize_draft_json` | 草稿内 `updatedAt` | 授权前可覆盖 |

**读侧规则**：`content.service` / H5 `case-render` / Feed 正文与过程节点 → **snapshot**；页顶摘要、FAQ、TDK → **enrichment 优先**，不得与 snapshot 事实矛盾。

---

## 2. `content_json.snapshot` 主要字段

| 字段 | 说明 |
| --- | --- |
| `version` | ≥1 表示有效快照；再授权递增 |
| `frozenAt` | ISO 冻结时间 |
| `authorizationTier` | `named` / `anonymous` 等 |
| `title` / `summary` / `articleBody` | 案例文案真源 |
| `coverImage` | 脱敏封面 URL |
| `nodes[]` | `{ id, title, note, images[] }` — **images 均为脱敏 URL** |
| `vehicle` / `parts` / `planParts` / `evidenceItems` | 授权瞬间留档 |
| `planAmount` / `price` | 方案价与展示价结构 |
| `geo` | 冻结时 GEO 叙事块（故障/检测/方案/结果） |
| `storeId` / `storeName` / `serviceName` / `city` | 门店与服务上下文 |
| `templateId` / `serviceItemId` | 模板与服务商品 |

提取：`extractSnapshotFromContentJson(contentJson)` · 节点读侧：`resolvePublicCaseContentNodes(contentJson)`

---

## 3. `enrichment_json` 主要字段

| 字段 | 说明 |
| --- | --- |
| `version` | 提炼层版本；变更 **不** bump snapshot |
| `aiSummary` | 页顶可引用摘要（可含聚合 N=） |
| `seoTitle` / `seoDescription` | TDK |
| `faq[]` / `faqLinks[]` | 页内 FAQ 与延伸阅读 |
| `schemaGraph` | JSON-LD |
| `topicMountIds[]` | 挂载 `geo_pages` |
| `slug` / `canonicalPath` / `seoNoindex` | URL 与收录控制 |
| `geo` | 发布态镜像（`publishedH5At` 等） |

守卫：`assertSnapshotPreserved` — enrichment 写入不得修改 `content_json.snapshot` 域。

---

## 4. `albums.content_optimize_draft_json`

| 字段 | 说明 |
| --- | --- |
| `status` | `ready` / `applied` / `failed` |
| `source` | `rule` / `llm_v1`（299 套餐） |
| `aiSummary` / `geo` / `nodeNotes` | 建议文案 |
| `suggestions[]` | 规则提示 |

授权时：`mergeOptimizeDraftIntoCaseDraft` 并入 case draft → 进入 snapshot；**不**在 optimize 阶段写 `public_cases`。

---

## 5. 状态与 API 对照

| 动作 | snapshot | enrichment | 相册 editable |
| --- | --- | --- | --- |
| 用户授权提交 | 创建/覆盖 | 可选初始化 | 409 |
| 运营 B 通过 | 不变 | 可写发布字段 | 409 |
| PUT enrichment | 守卫拒绝修改 | 更新 | — |
| LLM adopt（运营） | 409 if frozen | — | — |
| 用户撤回 | 保留 audit | — | 解锁 |
| 举报下线 | 保留 snapshot + `case_review_log` | — | — |
| 再授权 | version++ | 可合并 | 409 |

错误码：`SNAPSHOT_FROZEN` · `LLM_ADOPT_SNAPSHOT_FROZEN` · `ALBUM_CONTENT_LOCKED`

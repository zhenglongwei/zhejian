## **2. 模块定位**

V2.0 平台**不是**交易方，**不提供**订单退款、售后仲裁、维修质量担保。

本模块用于：

1. FAQ 自助（咨询方式、相册查看、授权公开、隐私说明）；
2. 联系平台（内容/隐私/合规类问题）；
3. 投诉举报（虚假案例、隐私泄露、违规宣传）；
4. 咨询线索问题反馈（商家未联系等，平台仅协助转达，不介入线下维修）。

**不做**：订单客服、退款申请、售后进度、评价奖励争议。

## **3. 入口**

| 入口 | 场景 |
| --- | --- |
| 我的 → 帮助与反馈 | 通用 |
| 案例/相册页 → 举报 | 内容/隐私 |
| 咨询记录 → 问题反馈 | 线索未响应 |
| 门店页 → 电话/咨询 | 优先引导联系门店（线下服务） |

---

## **4. 帮助中心 FAQ（P0）**

- 平台做什么、不做什么（非交易平台声明）；
- 如何提交咨询/预约；
- 如何查看服务相册、配件确认；
- 如何授权/撤回公开案例；
- 隐私与脱敏说明；
- 事故车/复杂维修为何不能线上报价。

---

## **5. 投诉类型**

| 类型 | 处理 |
| --- | --- |
| 隐私泄露 | 先隐藏内容，运营审核 |
| 未授权公开 | 下架案例，核查授权记录 |
| 虚假/违规案例 | 审核/下架，标记商家风险 |
| 咨询线索问题 | 记录并通知商家，平台不判定维修结果 |
| 配件/维修质量 | **引导联系门店或行政/司法途径**，平台不提供赔偿 |

---

## **6. 平台客服可操作（运营端）**

查看投诉、临时隐藏案例/相册、联系用户/商家、添加备注、关闭工单。

**不可做**：发起退款、修改订单状态、判定维修责任。

---

## **7. 消息通知**

投诉提交/处理结果通知用户；严重违规通知商家。见 `07_消息通知服务设计.md`（V2.0）。

---

## **8. P0 验收标准**

1. 帮助中心含平台身份与非交易声明；
2. 可提交隐私/内容类投诉；
3. 投诉可关联案例/相册/线索 ID；
4. 隐私类投诉可先触发隐藏；
5. 维修质量类展示「联系门店」引导，无退款入口。

## **2. 模块定位**

审核商家为**线下维修**主动创建的**服务相册**（非平台订单相册）。

主要服务于：

1. 维修过程可信记录与车主查看；
2. 配件告知与车主确认留痕抽查；
3. 争议场景证据留存（平台不仲裁线下纠纷）；
4. 公开案例生成前的合规 gate；
5. 商家内容质量与风控识别。

## **3. 相册来源**

| 来源 | 说明 |
| --- | --- |

## **4. 审核对象**

| 对象 | 审核内容 |
| --- | --- |
| 六阶段节点 | 接车、检测、方案、配件、施工、交付是否满足风险等级要求 |
| **配件确认** | 类型是否标准、非原厂是否发起确认、车主确认留痕 |
| 凭证照片 | 新旧件、包装、编码等（L2+） |
| 图片 | 清晰度、真实性、隐私、违规内容 |
| 文案 | 是否夸大、是否含隐私、是否暗示平台报价 |
| 服务关联 | 是否与服务项目、风险等级 L1–L4 一致 |
| 公开适用性 | 是否适合生成公开案例 |

## **5. 六阶段与节点要求**

对齐 `docs/04_维修过程相册/02_相册模板与节点规则.md`：

| 阶段 | 审核要点 |
| --- | --- |
| 一 接车 | 车辆/部件初始状态 |
| 二 检测 | L2+ 复杂服务须有检测记录 |
| 三 方案 | 门店参考说明，非平台报价 |
| 四 配件 | **配件类型 + 确认卡**；L2+ 非原厂须已确认 |
| 五 施工 | 施工过程、新旧件对比 |
| 六 交付 | 完工效果，无隐私单据 |

不同服务类型可在模板层细化（保养/钣喷/事故车等），L1 建议不强制，L2+ 按阻断规则。

## **7. 审核流程**

### **7.1 标准流程**

```text
商家上传/更新服务相册
→ 系统校验节点与配件状态
→ 自动隐私检测
→ 命中风险则进入审核队列
→ 平台审核
→ 通过 / 要求修改 / 驳回
→（可选）生成公开案例草稿
```

### **7.2 公开案例前**

```text
相册合规 + 用户授权
→ 自动脱敏
→ 生成案例草稿
→ 提交公开案例审核（见 04）
```

### **7.3 风险流程**

命中隐私/配件/造假风险 → 风险锁定 → 人工复核 → 退回脱敏 / 要求修改 / 禁止公开。

---

## **8. 审核重点（摘要）**

### **8.1 图片与隐私**

敏感信息：车牌、VIN、人脸、证件、保单、维修单据等。轻微可脱敏；严重则风险锁定。

### **8.2 六阶段与配件**

- L2+ 缺检测、缺凭证、**未配件确认即完工** → 不通过或标记风险；
- 只有完工图无过程 → 要求补传；
- 图片与服务项目明显不符 → 驳回。

### **8.3 真实性与风控**

多相册图片重复、复杂维修无过程、文案夸大（「100%修好」「全网最低」）、外部联系方式 → 风控。

### **8.4 是否适合公开**

须：清晰、节点够、已脱敏、用户授权（若需）、无配件/隐私争议、价格用区间表达。

## **9. 审核操作**

| 操作 | 说明 |
| --- | --- |
| 通过 | 相册合规，可留痕与邀请车主查看 |
| 要求修改 | 补节点、补凭证、修正配件类型 |
| 驳回 | 不符合最低要求 |
| 退回脱敏 | 隐私可处理 |
| 标记不可公开 | 可内部留痕，不可转案例 |
| 标记风险 | 生成风控事件 |
| 临时隐藏 | 投诉优先 |
| 生成案例草稿 | 满足授权与脱敏时 |

---

## **10. 驳回原因（示例）**

| 类型 | 示例 |
| --- | --- |
| 节点缺失 | 缺接车/交付或 L2+ 缺检测 |
| 配件不合规 | 非原厂未确认、类型未填 |
| 凭证缺失 | 缺新旧件/包装/编码照片 |
| 隐私未脱敏 | 含车牌/VIN/人脸 |
| 内容不真实 | 图片与服务不符、重复盗图 |
| 违规宣传 | 夸大承诺、外部联系方式 |
| 不适合公开 | 事故隐私、用户拒绝授权 |
| 风控命中 | 配件争议、隐私投诉 |

## **11. 后台功能要求**

1. 相册审核列表（按门店/服务/状态/风险筛选）；
2. 查看六阶段节点与配件确认记录；
3. 查看原图与脱敏图；
4. 查看用户授权状态；
5. 审核通过/驳回/要求修改/退回脱敏/标记不可公开；
6. 审核日志。

**删除**：按订单筛选、查看退款/售后状态。

## **13. 与风控联动**

| 场景 | 处理 |
| --- | --- |
| L2+ 未配件确认即完工 | 标记相册/商家风险 |
| 复杂维修无检测图 | 人工审核 |
| 图片含车牌/VIN | 内容风险，禁止公开 |
| 多相册图片重复 | 商家风险 |
| 隐私投诉 | 先隐藏后审核 |

---

## **14. P0 验收标准**

1. 平台可查看服务相册列表与六阶段内容；
2. 可查看配件确认留痕；
3. 可查看原图/脱敏图并完成隐私审核；
4. 支持通过、驳回、要求修改、退回脱敏、标记不可公开；
5. 高风险进入风控队列；
6. 审核操作留痕；
7. 含隐私图片不得直接公开。

## **2. 模块定位**

主要覆盖：

1. 商家入驻真实性；
2. 服务方案展示合规性；
3. **服务相册**真实性与配件告知留痕；
4. 公开案例隐私与内容安全；
5. **咨询线索**异常（垃圾提交、骚扰）；
6. H5 / GEO 公开内容合规。

---

## **3. 风控对象**

| 对象 | 风险类型 |
| --- | --- |
| 用户 | 垃圾咨询、恶意投诉、隐私滥用、多账号骚扰 |
| 商家 | 虚假资质、违规报价、虚假案例、相册造假、配件未告知 |
| 服务相册 | 隐私未脱敏、节点缺失、配件未确认即完工、图片重复 |
| 公开案例 | 隐私泄露、虚假内容、侵权图片、未授权公开 |
| 咨询线索 | 虚假信息、刷单式提交、骚扰商家 |
| H5/GEO 内容 | 违规宣传、隐私公开、平台报价暗示、错误收录 |

---

## **4. 风险等级**

| 等级 | 说明 | 处理方式 |
| --- | --- | --- |
| R0 无风险 | 正常业务 | 自动通过 |
| R1 低风险 | 轻微异常 | 通过并记录 |
| R2 中风险 | 需关注 | 人工审核 |
| R3 高风险 | 可能违规 | 先隐藏/限制，再审核 |
| R4 严重风险 | 明确违规违法 | 拦截、下架、封禁 |

---

## **5. 用户风控**

### **5.1 异常规则**

1. 短时间大量重复提交咨询/预约；
2. 多次恶意投诉或虚假举报；
3. 多次申请案例下架且理由可疑；
4. 与商家存在疑似关联的异常行为；
5. 公开授权后频繁撤回并重复提交。

### **5.2 限制措施**

| 措施 | 说明 |
| --- | --- |
| 咨询限制 | 短期限制提交咨询 |
| 公开授权限制 | 暂停新授权申请 |
| 账号冻结 | 禁用部分功能 |
| 平台客服介入 | 人工核查 |

---

## **6. 商家风控**

### **6.1 异常规则**

1. 营业执照或资质疑似伪造；
2. 服务项目超出资质范围；
3. 事故车服务承诺线上最终价；
4. 上传案例疑似非本店案例；
5. **L2+ 服务相册未配件确认即标记完工**；
6. 多相册使用相同图片（造假）；
7. 复杂服务相册缺少检测/凭证节点；
8. 被用户投诉诱导、虚假宣传；
9. 投诉/举报率异常；
10. 相册完整度长期极低。

### **6.2 限制措施**

| 措施 | 说明 |
| --- | --- |
| 服务下架 | 违规服务不可展示 |
| 案例/相册下架 | 隐藏违规内容 |
| 门店降权 | GEO/推荐权重降低 |
| 限制新建相册 | 暂停创建服务相册 |
| 商家冻结 | 暂停工作台核心功能 |
| 清退 | 严重违规退出平台 |

---

## **7. 服务相册风控**

### **7.1 异常规则**

1. 图片含车牌/VIN/人脸/证件未脱敏；
2. 非原厂/同质件未发起车主确认即完工（L2+）；
3. 缺少新旧件、包装、编码等关键凭证（L2+）；
4. 六阶段关键节点严重缺失 — **商家留档风控/提醒**，**不** 作为用户授权公示或用户授权案例运营通过的硬门槛；
5. 同一图片用于多个相册；
6. 文案含「100%修好」「全网最低」等违规承诺；
7. 用户隐私投诉关联相册。

### **7.2 限制措施**

标记风险、退回脱敏、禁止公开、禁止转案例、临时隐藏、生成风控事件。

详情联动 `03_服务相册审核.md`。

---

## **8. 咨询线索风控**

### **8.1 异常规则**

1. 同一用户/设备短时间大量提交；
2. 明显虚假手机号或无效描述；
3. 重复向多门店发送相同垃圾内容；
4. 含违规广告、外链、辱骂内容。

### **8.2 处理**

标记异常、限制提交、内部备注；**平台不介入线下维修纠纷与资金问题**。

---

## **9. 内容与隐私风控**

### **9.1 文本隐私检测**

车牌、手机号、VIN、身份证、详细住址、保险单号、微信号、银行卡号等。

### **9.2 图片隐私检测**

车牌、VIN 铭牌、行驶证、驾驶证、身份证、保险单、人脸、客户资料等。

### **9.3 案例公开要求**

1. 用户授权或商家历史案例责任确认；
2. 业务链路须完成脱敏（**不等于**审核员在审核台查看原图；审核员仅看脱敏图 + OCR）；
3. 价格使用区间或影响因素，**不得暗示平台报价**；
4. 标明案例来源（用户授权 / 商家上传）；
5. 用户可申请下架；
6. H5 收录需单独判断（noindex 规则）。

---

## **10. H5 / GEO 合规风控**

### **10.1 禁止展示**

未授权档案、未脱敏图片、隐私信息、**平台统一成交价承诺**、夸大宣传、虚假原厂授权。

### **10.2 noindex 规则**

隐私风险中等、稀有车型识别度高、用户不愿收录、案例质量不足、配件/隐私争议未结案。

---

## **11. 禁止行为（合规底线）**

1. 好评返现、晒图返现、分享领钱；
2. 「全网最低」「100% 修好」；
3. 事故车线上最终报价；
4. 诱导绕过平台展示规则的违规导流；
5. 未脱敏公开用户隐私；
6. 非原厂件未告知却对外宣传为原厂。

---

## **12. 风控处理流程**

```text
业务动作触发 → 规则检测 → 风险等级 → 通过/记录/审核/隐藏/拦截 → 风控日志 → 通知
```

人工审核：进入风险队列 → 查看关联用户/商家/相册/案例 → 处理并留痕。

入口：`平台运营工作台 → 风控中心`（MVP 可合并在各审核模块）。

Tab 建议：用户风险、商家风险、**相册风险**、内容风险、**线索风险**。

## **14. 数据结构建议**

**risk_event**：risk_type 含 `user` / `merchant` / `album` / `case` / `lead` / `content`（**无** order / refund / reward）。

**risk_rule_config**：规则参数可配置开启/关闭。

---

## **15. MVP 风控规则清单（P0）**

1. 图片含车牌/VIN/人脸须脱敏后才可公开；
2. 文本含手机号/VIN 进入审核；
3. 未授权案例不可公开；
4. 事故车不得线上承诺最终价；
5. L2+ 非原厂件未车主确认 → 标记相册风险；
6. 商家资质不匹配服务不可上架；
7. H5 未脱敏内容不可发布；
8. 重复/垃圾咨询提交限流或标记；
9. 相册图片跨相册重复 → 商家风险；
10. 用户撤回授权 → 案例紧急下线。

---

## **16. P0 验收标准**

1. 支持用户、商家、相册、案例、线索、内容的基础风控规则；
2. 规则可配置开启/关闭；
3. 风险事件可记录与查询；
4. 中高风险进入平台审核；
5. 高风险内容可先隐藏再审核；
6. 案例公开前须通过隐私检查；
7. H5 仅展示脱敏且已授权/已审核内容；
8. 处理操作留痕；
9. **不支持**订单/支付/退款/奖励类风控。

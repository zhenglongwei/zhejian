## 1. 环境与数据准备

### 1.1 真机环境

参考 `B-INF_生产与真机联调.md`：

- [ ] 体验版/预览版 **关闭**「不校验合法域名」
- [ ] 测试账号已登录且 **已绑定手机号**（与门店登记一致）
- [ ] 从 **我的 → 我的服务相册 → 点击卡片** 进入详情（非仅开发者工具模拟）

### 1.2 建议测试相册（至少覆盖 4 种）

| 编号 | 场景 | `publicCaseStatus` / 条件 | 用途 |
| ---: | --- | --- | --- |
| A | 多图多节点 | 私密 · 已完工 · ≥6 张 · ≥3 节点 | 翻页 / Tab / 抽屉主路径 |
| B | 可授权未公示 | `private` 或 `authorization_pending` · 已完工 · 有图 | 尾页「授权公示」+ 授权流程 |
| C | 审核中 | `review_pending`（列表 Tag「审核中」） | 尾页「审核中」弱态 + 说明文案 |
| D | 已公开 | `public_approved` | 尾页分享 + 公开案例链接 |
| E | 无过程图 | `imageCount = 0` | empty 态 |
| F | 无权限 | 他人相册 ID 或错误 `albumId` | error 态 · 403 |

> 若 staging 暂无 F，可用错误 `albumId` 模拟；B/C/D 可同一相册在不同阶段复测。

### 1.3 入口路径

- [ ] **主路径**：我的 → 我的服务相册 → 点击卡片主体
- [ ] **返回**：详情 ← 返回 → 回到列表，状态不异常

## 2. 页面态（§3.1）

### 2.1 loading

- [ ] 进入详情首屏：沉浸导航可见，相框区为 **Skeleton**（非空白闪屏）
- [ ] loading 期间 **无** 分享/授权入口误露

### 2.2 normal（有图）

- [ ] 加载完成后：相框 + 底栏 Tab + 页脚同时出现
- [ ] **无** 置顶摘要卡、**无** 服务/门店/车辆四行表

### 2.3 empty（相册 E）

- [ ] 文案含「门店暂未上传服务相册」类提示
- [ ] 有「联系门店」操作
- [ ] 页脚仍展示 **联系门店 + ComplianceNotice**
- [ ] **无** 翻页相框、**无** 分享入口

### 2.4 error（相册 F / 未登录）

- [ ] 无权限：提示明确（如「仅关联车主可查看」），有 **重试**
- [ ] 未登录/未绑手机：LoginSheet 可拉起，成功后自动 reload
- [ ] error 态 **无** ShareSheet / 授权入口

- [ ] 系统状态栏不被内容遮挡；导航与胶囊 **不重叠**
- [ ] 左侧 **‹ 返回** 可用
- [ ] 标题为 **「服务相册」**
- [ ] 导航右侧 **无** 分享按钮（分享仅在尾页 + 列表卡片）

## 4. 相框 `AlbumPhotoFrame`（variant=detail）

- [ ] 四边等宽相框；角饰 L 形可见（非纯矩形白边）
- [ ] 立体阴影 `--shadow-album-frame` 有层次感（非扁平贴图）
- [ ] 档案纸底 `--color-album-paper` 与页面 `bg-album` 协调
- [ ] **无** 促销 Banner、**无** 大面积主色块（主色面积 ≤15% 主观检查）

---

## 5. 翻页 `AlbumFrameViewer`

### 5.1 框内顶栏

- [ ] 左侧页码格式 **`{当前} / {总数}`**，总数 = **图片张数 + 1（尾页）**
- [ ] 右侧 **「信息」** 可点击，打开抽屉（§8）
- [ ] **无** 分享按钮

### 5.2 翻页交互

- [ ] 左右滑动切换图片；**无** 露边预览、**无** 大箭头按钮
- [ ] 点击图片 **不** 触发 `previewImage` / 系统全屏预览
- [ ] 长按图片 **不** 出现保存菜单（`show-menu-by-longpress=false`）
- [ ] 滑到最后一页过程图后再滑一页 → 进入 **尾页**（§7）

### 5.3 框内底文案条（仅图片页）

- [ ] 显示：**节点名 · 时间**（有 note 时含摘要，单行省略）
- [ ] 尾页 **不显示** 底文案条
- [ ] **无**「对此节点有疑问」链入框内（反馈仅在尾页）

### 5.4 图片加载

- [ ] 各节点图片正常显示（非灰块 / 非裂图）
- [ ] 弱网下图标/占位合理，不崩溃

## 6. 节点 Tab `AlbumBottomToolbar`

- [ ] 底栏位于相框下方、页脚上方；半透明可辨
- [ ] 展示各节点标题；**当前节点高亮**
- [ ] 节点较多时可 **横滑** Tab
- [ ] 点击 Tab → 跳到该节点 **第一张** 过程图
- [ ] 在尾页时 Tab 仍可见且可点击跳转
- [ ] **无**「授权公示」按钮
- [ ] **无**「联系门店」（联系在页脚）

---

## 7. 尾页 `AlbumEndPage`

### 7.1 布局

- [ ] 无大图；暖色档案纸底 + 相框内留白
- [ ] 页码为最后一页（如 `13 / 13`）

### 7.2 按钮（相册 B / C / D）

| 按钮 | 相册 B（可授权） | 相册 C（审核中） | 相册 D（已公开） |
| --- | --- | --- | --- |
| 授权公示 | ☐ 显示「授权公示」可点 | ☐ 显示「审核中」禁用 + 说明 | ☐ 不显示或已隐藏 |
| 分享 | ☐ 按 `canOwnerShare` | ☐ | ☐ 显示且可开 ShareSheet |
| 反馈 | ☐ 始终显示 | ☐ | ☐ |

- [ ] 点击 **反馈** → 进入 `pages/album/feedback`，携带 `albumId`

## 8. 信息抽屉 `AlbumInfoSheet`

- [ ] 标题 **「说明与隐私」**；右上角可关闭
- [ ] 面板高度约半屏（55–65%），内容可纵向滚动
- [ ] 顶部 **PrivacyBanner**（scene=album）
- [ ] **门店说明**列表：有 `note` 的节点展示缩略图 + 节点名 + **全文**（不截断）
- [ ] 若有 `storeNote`，置顶 **门店总说明**
- [ ] **不在抽屉内**：摘要四行表、Tag、授权表单、分享、反馈、ComplianceNotice
- [ ] 点击遮罩关闭抽屉

---

## 9. 页脚 `AlbumPageFooter`

- [ ] 固定在页面最底部（Tab 下方），含 safe-area
- [ ] **联系门店** 按钮：有电话时调起拨号；无电话 Toast 提示
- [ ] **ComplianceNotice** 浅条展示（详情页 **唯一位**，主屏无 PrivacyBanner）
- [ ] 文案符合平台合规表达，无诱导分享/返现

---

## 10. 授权流程（相册 B）

- [ ] 尾页点「授权公示」→ 打开 BottomSheet「授权公示」
- [ ] 含勾选确认、`ui-album-authorize-section`（实名/匿名 tier）
- [ ] 未勾选点提交 → Toast「请先勾选确认项」
- [ ] 「确认并进入脱敏预览」→ 跳转脱敏预览页（`pages/desensitize/preview`）
- [ ] 预览页脱敏图可加载（车牌等已打码，见 A-MASK 口径）
- [ ] 「拒绝公示」→ 二次确认 → 提交成功 Toast
- [ ] 提交后返回详情，状态刷新（尾页/列表 Tag 同步，若可测）

---

## 11. 分享流程（相册 D 或 B 可分享）

- [ ] 尾页点「分享」→ 打开 `ShareSheet`
- [ ] 车主分享：默认脱敏模式；切换原图有 **风险提示** Modal
- [ ] 「复制链接」成功；链接可在浏览器/H5 打开（若环境允许）
- [ ] 「分享到朋友圈」引导 Modal（confirmText ≤4 字）
- [ ] 已公开相册：可复制 **公开案例 Web 链接**（若有）
- [ ] 分享文案 **无**「须先授权才能分享」「分享即公开」等混用
- [ ] 分享文案 **无** 诱导现金/返现

## 12. 权限与安全（P0）

- [ ] 相册 F：403 不可查看他人相册
- [ ] 未登录不可查看（LoginSheet）
- [ ] 未绑手机不可查看（bindPhone 模式）
- [ ] error/empty 态不出现分享菜单（`wx.hideShareMenu`）
- [ ] 详情主屏 **不** 暴露 rawUrl / 未脱敏直链（C 端口径）

## 13. 禁止项负向验收（§3.10 · PRD §14.2）

以下 **任一出现即判不通过**：

- [ ] 详情页出现 **置顶摘要卡** / ServiceAlbumSummaryCard
- [ ] 框内顶栏出现 **分享**
- [ ] 主屏（相框外）出现 **PrivacyBanner**（仅抽屉内允许）
- [ ] 底栏出现 **授权公示** 或 **联系门店**
- [ ] 点击图片进入 **previewImage** 全屏
- [ ] 出现促销模块 / 好评返现 / 分享领钱类文案
- [ ] 对用户展示内部态文案「可授权」「已授权」「已公开」（应用 Phase 1 用户可见文案）

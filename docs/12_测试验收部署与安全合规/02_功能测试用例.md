## **2. 用例等级说明**


| 等级  | 说明                 | 是否阻塞上线 |
| --- | ------------------ | ------ |
| P0  | 核心咨询线索、相册、隐私、安全、权限相关 | 是      |
| P1  | 主要业务功能异常，影响核心体验    | 原则阻塞   |
| P2  | 非核心功能或体验问题         | 可评估延期  |
| P3  | 文案、样式、轻微体验问题       | 不阻塞    |

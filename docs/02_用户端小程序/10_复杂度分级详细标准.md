## **2. 分级原则**

1. **技术难度优先**：对技师资质、设备、诊断能力要求越高，等级越高。
2. **风险绑定**：资金与车辆安全风险越高，等级越高，相册与配件确认要求越严。
3. **资质匹配**：等级对应《机动车维修管理规定》法定维修资质；用户端可提示「建议由具备相应资质门店承接」，**不**做平台派单过滤（线下由门店与用户自行确认）。
4. **就高不就低**：一次服务涉及多项目时，相册与展示按**最高等级**执行（商家创建相册时选择/系统建议）。

## **3. 四级标准概览**

| 等级 | 名称 | 典型客单 | 相册要求（摘要） | 建议资质 |
| --- | --- | --- | --- | --- |
| **L1** | 极低复杂度 | ＜500 元，＜1h | 普通记录，拍照可选 | 三类及以上 |
| **L2** | 低复杂度 | 500–3000 元，1–4h | 配件类型必填；非原厂须车主确认；**未确认不可完工** | 二类/三类专项 |
| **L3** | 中复杂度 | 3k–1 万，4–24h | 方案确认 + 配件确认 + 新旧件对比；阻断完工 | 二类及以上 |
| **L4** | 高复杂度 | 1 万+，3 天+ | 强制方案/配件/验收确认；证据包；建议电子签名 | 一类 |

相册细则见 `docs/04_维修过程相册/02_相册模板与节点规则.md`、`10_配件告知确认规则.md`。

## **4. L1 · 极低复杂度**

### **4.1 定义**

无技术门槛或仅超低价易损件，无车辆安全风险，便民类服务。

### **4.2 覆盖示例**

洗车、内饰清洁、打蜡/封釉；轮胎补气、动平衡（不含四轮定位）；空调/空气滤芯更换（无需拆核心件）；雨刮、灯泡、保险丝；基础安全检测、读码（不含排查维修）。

### **4.3 特征**

施工简单，普通作业人员可完成。

### **4.4 V2.0 平台规则**

| 维度 | 规则 |
| --- | --- |
| 用户端展示 | 服务/案例可标 `L1` 或「基础服务」；价格可用一口价/起步价 |
| 服务相册 | 六阶段可简化；节点多为 **建议** 非强制；无配件确认阻断 |
| 案例公开 | 常规脱敏审核即可 |
| 资质提示 | 建议三类及以上（展示用，非平台担保） |

---

## **5. L2 · 低复杂度**

### **5.1 定义**

标准化更换/维保，安全风险低，需基础维修技能与专用工具。

### **5.2 覆盖示例**

小保养；刹车片/盘/油；火花塞、燃油滤、重力换变速箱油；电瓶、轮胎、四轮定位；单块翼子板/车门小钣喷（无结构件）；常规易损件；新能源常规保养、非三电易损件。

### **5.3 特征**

标准化流程，无核心总成拆解。

### **5.4 V2.0 平台规则**

| 维度 | 规则 |
| --- | --- |
| 用户端展示 | 参考区间或套餐价 + 影响因素；案例标 `L2` |
| 服务相册 | 须记录配件类型；**非原厂件须车主确认**；存在 `pending` 配件确认时 **不可标记完工** |
| 必传建议 | 新旧件对比、施工结果（见相册 PRD） |
| 资质提示 | 二类 / 三类专项 |

---

## **6. L3 · 中复杂度**

### **6.1 定义**

需专业诊断与核心部件局部拆解，存在一定安全与资金风险。

### **6.2 覆盖示例**

发动机/变速箱局部维修（非总成大修）；底盘整备（减震、下摆臂、转向机等）；制动深度维修；车身电气深度维修；中大面积钣喷（≥3 面，无结构件）；新能源三电基础检测、非核心三电部件、电池均衡维护等。

### **6.3 特征**

需诊断设备与中级技师，涉及核心系统拆解。

### **6.4 V2.0 平台规则**

| 维度 | 规则 |
| --- | --- |
| 用户端展示 | **参考区间**或「到店检测后确认」；须展示价格影响因素 |
| 服务相册 | **维修方案确认**（车主勾选）；配件确认 + 新旧件/包装/编码图；方案未确认不可录入施工完成 |
| 交付 | 建议 **交付验收确认** |
| 资质提示 | 二类及以上 + 对应持证技师 |

---

## **7. L4 · 高复杂度**

### **7.1 定义**

总成大修、整车/事故车结构修复、疑难故障，资金与安全风险极高。

### **7.2 覆盖示例**

发动机/变速箱总成大修；事故车整车修复（结构件、纵梁/ABC 柱、整车喷漆）；泡水车、整车电路重构；疑难多系统故障；新能源三电总成大修、整车事故修复、三电疑难故障。

### **7.3 特征**

长周期、高级技师与成套设备，法定资质要求高。

### **7.4 V2.0 平台规则**

| 维度 | 规则 |
| --- | --- |
| 用户端展示 | **仅预约/到店检测**；禁止线上最终价；须事故车/复杂维修免责声明 |
| 服务相册 | 强制方案确认、配件确认、证据包；高风险配件二次确认；建议电子签名与验收确认 |
| 案例 | 事故车仅展示参考区间与影响因素 |
| 资质提示 | 一类维修资质 + 高级技师与专项设备 |

## **8. 兜底与判定规则**

### **8.1 多项目就高**

同一服务相册关联多项维修时，按 **最高 L 级** 套用相册节点与配件确认规则。

### **8.2 模糊项目**

未在清单中的项目，按技术难度、典型客单、安全风险三维度由系统建议等级，商家可调整并留痕；争议由运营/相册审核复核。

### **8.3 资质提示（非派单）**

用户端与 GEO 页可展示「该项目复杂度较高，建议选择具备 XX 资质门店」——**不构成**平台派单、担保或拒绝展示。

### **8.4 新能源**

涉及三电：**最低 L3**；三电总成大修：**L4**；须新能源专项资质方可对外宣传相关能力（商家审核侧校验）。

## **9. 与用户端展示的关系**

| 页面 | 用法 |
| --- | --- |
| 服务详情 | 可选展示复杂度标签；L3/L4 强化「到店检测」与影响因素 |
| 案例卡片 | 可选 `L2`/`L3`/`L4` 标签，辅助用户判断参考性 |
| 门店详情 | 不单独按 L 级排序；通过擅长项目与案例体现 |
| 搜索筛选 | P1 可按复杂度筛选案例 |

## **11. P0 验收要点**

1. 服务/相册创建时可选择或系统建议 L1–L4；
2. L2+ 配件 pending 时相册不可 `completed`（与相册 PRD 一致）；
3. L3/L4 用户端不出现线上最终价；
4. 用户端不出现与 L 级绑定的奖励/评价/竞价文案；
5. 多项目相册按最高等级阻断规则生效。

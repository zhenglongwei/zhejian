## **2. 我的咨询**

（同 V2.0，无变更）

**状态（简化）**：已提交、门店已查看、门店已联系、用户已取消、已关闭。

**不做**：订单态、支付、履约。

## **3. 我的服务相册**

用户可查看门店为其创建的 **服务相册**（六阶段时间线）。

**来源**：门店关联手机号；分享链接；扫码（P1）。

**列表 Tab（Phase 1）**：**私密相册** / **公开相册**（仅两 Tab，无「全部/进行中/已完工/待授权」）。

| Tab | 筛选规则 |
| --- | --- |
| **私密相册** | 尚未公开展示（`publicCaseStatus !== public_approved`），含维修中、已完工未申请、**审核中**、撤回后私密 |
| **公开相册** | 已审核通过并公开展示（`public_case` / `publicCaseStatus === public_approved`） |

**列表卡片 Tag（用户端）**：

| Tab | 卡片 Tag |
| --- | --- |
| 私密相册 | **维修中** 或 **已完工**；若处于平台审核中，额外展示 **审核中** |
| 公开相册 | 不展示维修/审核 Tag（Tab 本身已表达公开语义） |

**禁止**在列表使用面向内部的文案：「可授权」「已授权」「已公开」「待授权公开」等。

详情页头部仍展示 **维修进度**（维修中/已完工）+ **可见性**（私密相册/审核中/公开相册）双层 Tag，见 `07_服务相册查看页.md` §14.1。

**列表字段**：门店、脱敏车辆、服务项目、更新时间、过程图张数。

**Phase 1 不做**：配件/方案待确认主路径（Phase 2）。

详情与授权公开见 `07_服务相册查看页.md`。

## **4. 我的公开授权**

用户已授权公开的档案列表；可查看审核状态；**任何时候** 均可撤回（含已审核通过并已公示的案例，见 `12_我的页面与账户体系.md` §8.1）。

---

## **5. 与咨询的关系**

咨询记录与服务相册 **解耦**：咨询不产生订单；门店线下维修后 **单独创建服务相册** 并可关联同一手机号。

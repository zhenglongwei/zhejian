## **3. 我的服务相册**

用户可查看门店为其创建的 **服务相册**（六阶段时间线）。

**来源**：门店关联手机号；分享链接；扫码（P1）。

**列表 Tab（2026-06-24 修订）**：**全部** / **可公示** / **已公示**（三 Tab；**不再**单独保留「授权公示」页）。

| Tab | 筛选规则 |
| --- | --- |
| **全部** | 所有服务相册 |
| **可公示** | 尚未进入公示流程或已退回私密（含维修中、已完工未申请、用户拒绝、审核未通过/撤回后）；**不含** `pending_review` / `public_approved` |
| **已公示** | 已提交授权且在审核中或已通过（`pending_review`、`public_approved`） |

**列表卡片 Tag（用户端）**：

| Tab | 卡片 Tag |
| --- | --- |
| 全部 / 可公示 | **维修中** 或 **已完工** |
| 已公示 | 不展示维修进度 Tag（Tab 本身表达公示语义） |

**卡片操作**：可公示 Tab 展示 **授权公示**；已公示 Tab 展示 **撤回公示**（含审核中与已通过，随时可撤回）。

**禁止**在列表使用面向内部的文案：「可授权」「已授权」「已公开」「待授权公开」等。

详情页仍按 `07_服务相册查看页.md` §14 展示维修进度与授权/分享操作。

**列表字段**：门店、脱敏车辆、服务项目、更新时间、过程图张数。

**Phase 1 不做**：配件/方案待确认主路径（Phase 2）。

详情与授权公开见 `07_服务相册查看页.md`。

## **15. 维修档案**

### **15.1 档案定位**

订单完成后，平台可生成维修档案。

维修档案包含：

1. 订单基础信息；
2. 服务项目；
3. 门店信息；
4. 维修进度；
5. 维修相册；
6. 用户评价；
7. 售后记录，可选；
8. 是否允许公开。

### **15.2 档案状态**


| 状态             | 说明      |
| -------------- | ------- |
| generating     | 生成中     |
| generated      | 已生成     |
| private        | 私密档案    |
| public_pending | 申请公开审核中 |
| public         | 已公开     |
| rejected       | 公开审核未通过 |


### **15.3 公开授权**

维修档案默认私密。

公开前必须：

1. 用户授权；
2. 平台脱敏；
3. 平台审核；
4. 不展示敏感信息；
5. 不展示实际支付金额，除非规则明确允许且合规。

## **22. 隐私与安全**

1. 订单详情仅订单本人可查看；
2. 用户手机号脱敏展示；
3. 车牌脱敏展示；
4. VIN 不完整展示；

6. 维修档案默认私密；
7. 公开案例需单独授权和脱敏审核；
8. 分享订单详情时，不得分享私密订单页面；
9. 用户分享公开案例时，使用脱敏公开页。

## **24. 合规要求**

1. 私密订单不得被公开访问；
2. 维修档案默认私密；
3. 公开前必须授权和脱敏；
4. 事故车订单持续展示“检测后报价”说明；
5. 不展示诱导分享奖励文案；
6. 不将维修相册自动作为公开案例；
7. 不展示用户完整手机号、车牌、VIN；
8. 费用和退款状态必须准确；
9. 售后入口不得隐藏或误导用户。

10. 保险定损记录。

## **29. 验收标准**

1. 用户可查看自己的订单列表；
2. 用户不可查看他人订单；
3. 订单状态展示正确；
4. 不同状态展示正确操作按钮；
5. 用户可查看维修进度；
6. 用户可进入维修相册；
7. 用户可联系门店；
8. 用户可导航到门店；
9. 用户可确认完工；
10. 完成订单后可进入评价；
11. 符合条件的订单可申请售后；
12. 车牌、手机号、VIN 均脱敏展示；
13. 事故车订单不展示线上最终价；
14. 维修档案默认私密；
15. 页面无诱导分享奖励文案。

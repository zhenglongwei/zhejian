## 2. 模块定位

门店详情页是用户选择维修服务承接方的核心页面。

该模块的目标是：

1. 展示门店真实信息；
2. 展示门店资质和平台审核状态；
3. 展示门店擅长项目；
4. 展示门店服务方案；
5. 展示门店公开案例；
6. 展示用户评价；
7. 展示门店环境和维修能力；
8. 引导用户预约、下单、导航或联系门店；
9. 避免未经审核的品牌授权、资质、宣传语误导用户。

## **11. 门店资质展示**

### **11.1 必须展示**

门店详情页必须展示：

1. 营业执照认证状态；
2. 维修资质认证状态；
3. 门头照片；
4. 工位照片；
5. 门店地址；
6. 平台审核状态。

### **11.2 资质字段**


| 字段      | 说明          |
| ------- | ----------- |
| 营业执照状态  | 已认证/认证中/未认证 |
| 维修资质类型  | 一类/二类/三类/专项 |
| 资质有效期   | 可选展示        |
| 平台审核时间  | 可选          |
| 门店真实性审核 | 已审核         |
| 负责人认证   | 可选          |
| 品牌授权    | 可选          |


### **11.3 品牌授权展示规则**

品牌授权不是入驻必要条件。

仅当商家提交并通过平台审核后展示。

展示文案：

```
品牌授权：已认证

```

未提交或未审核时不展示，不展示“无品牌授权”避免误伤门店。

### **11.4 禁止展示**

不得展示未经审核的：

1. 原厂授权；
2. 4S 同等；
3. 官方认证；
4. 品牌合作；
5. 全网最低；
6. 保证修复；
7. 保险指定合作，除非有证明。

## **15. 用户评价**

### **15.1 模块目标**

### **15.2 评价维度**

4. 维修过程透明；

### **15.3 评价卡片字段**


| 字段   | 说明           |
| ---- | ------------ |

| 用户头像 | 微信头像或默认头像    |
| 评分   | 1-5 星        |
| 评价内容 | 审核后展示        |
| 服务项目 | 可选           |
| 车辆信息 | 脱敏展示，可选      |
| 图片   | 用户上传图，需审核/脱敏 |
| 评价时间 | 可选           |
| 商家回复 | 可选           |


### **15.4 评价展示规则**

1. 仅展示审核通过评价；
2. 涉及隐私内容需脱敏；
3. 涉及争议评价可延迟展示；
4. 违规、辱骂、广告评价不展示；
5. 用户删除或平台下架后不展示。

### **15.5 评分计算**

门店评分可综合：

1. 服务态度；
2. 专业程度；
3. 价格透明；
4. 维修过程透明；
5. 完工效果；
6. 推荐意愿。

## **20. 异常与边界场景**

### **20.1 门店下架**

提示：

```
该门店暂不可查看。

```

或：

```
该门店暂不可预约，你可以查看其他门店。

```

### **20.2 门店冻结**

冻结门店不允许展示在列表和推荐位。

已打开详情页时提示：

```
该门店当前暂不可预约。

```

### **20.3 资质过期**

资质过期时：

1. 不展示资质认证标签；
2. 不允许平台推荐；
3. 可进入风控审核；
4. 必要时暂停预约。

### **20.4 门店无案例**

展示：

```
该门店暂无公开案例。

```

可引导查看服务或联系门店。

### **20.5 门店无评价**

展示：

```
该门店暂无评价，完成服务后可发表真实评价。

```

### **20.6 门店暂停营业**

展示：

```
该门店当前暂停营业，暂不可预约。

```

## **21. 权限与隐私**

### **21.1 用户定位**

定位仅用于：

1. 展示距离；
2. 门店排序；
3. 导航辅助。

不得公开用户位置。

### **21.2 用户评价隐私**

评价中涉及：

1. 车牌；
2. 手机号；
3. VIN；
4. 人脸；
5. 证件；
6. 住址；

必须脱敏或不展示。

### **21.3 门店资料隐私**

商家联系人手机号、身份证、后台资料不得展示在用户端。

---

## **22. 合规要求**

### **22.1 资质合规**

1. 营业执照状态必须真实；
2. 维修资质必须审核；
3. 品牌授权必须有凭证；
4. 不得展示虚假认证；
5. 资质过期应自动降级或隐藏。

### **22.2 宣传合规**

禁止展示：

1. 全网最低；
2. 绝对正品，未有证明；
3. 官方指定，未有证明；
4. 保证修好；
5. 永不复发；
6. 事故车线上精准报价；
7. 虚假案例数量；
8. 虚假评分。

### **22.3 案例合规**

门店案例必须：

1. 已审核；
2. 已脱敏；
3. 来源明确；
4. 价格区间展示；
5. 不展示实际支付金额；
6. 不展示隐私信息。

## **28. 验收标准**

1. 用户可进入门店列表页；
2. 用户可按服务查看支持门店；
3. 用户可进入门店详情页；
4. 门店详情展示名称、地址、营业时间；
5. 门店资质状态展示正确；
6. 未审核资质不展示认证标签；
7. 用户授权定位后可展示距离；
8. 用户拒绝定位不影响浏览门店；
9. 用户可点击导航；
10. 用户可点击联系门店；
11. 用户可查看门店服务方案；
12. 用户可从门店进入服务详情或下单预约；
13. 用户可查看门店公开案例；
14. 门店案例均为审核通过且已脱敏；
15. 用户可查看门店评价；
16. 门店下架后不出现在列表；
17. 门店冻结后不可预约；
18. 事故车相关服务不展示线上最终报价；
19. 门店分享卡片不包含诱导奖励文案；
20. 页面异常时有明确提示。

## **2. 模块定位**

承接首页、搜索、案例、门店、GEO 等入口的服务决策流程。

核心目标：

1. 让用户看懂服务项目与适用场景；
2. 让用户理解**价格模式**（一口价/套餐/参考区间/仅预约）及影响因素；
3. 让用户知道哪些门店可提供该服务；
4. 让用户查看类似脱敏案例；
5. 区分标准服务、价格区间服务、复杂维修与事故车；
6. 引导进入 **电话咨询 / 提交咨询 / 预约到店**（见 `05_咨询与到店预约.md`）；
7. 避免复杂维修与事故车线上误导报价。

## **8. 价格模式**

与设计体系 §6.1 四型对齐（事故车归入「仅预约」）。

| 模式 | 编码建议 | 展示 | 主 CTA |
| --- | --- | --- | --- |
| 一口价/套餐价 | fixed / package | 明确金额 + 包含项 | 预约到店 / 提交咨询 |
| 起步价 | start_price | ¥299 起 + 说明 | 同上 |
| 参考区间 | range_price | 参考区间 + 影响因素 | 预约到店 / 咨询报价 |
| 到店检测后报价 | inspection_quote | 到店检测后确认 | 预约到店检测 |
| 事故车 | accident_quote | 需到店检测/拆检后确认 | 预约到店 / 查看案例 |

### **8.1 一口价/套餐/起步价**

须展示：价格、包含/不包含、适用车型或限制、是否含配件/工时。

示例：

```text
套餐价 ¥399
包含：机油、机滤、基础检查、工时
不包含：额外维修、非套餐配件、特殊车型差价
```

**删除 V1.0**：是否支持退款、是否支持在线支付。

说明：页面标价为**参考说明**，最终费用以门店线下检测与确认为准。

### **8.2 参考区间**

```text
参考区间：¥300-¥1200
实际费用会因车型、配件品牌、损伤程度、工时和门店检测结果而不同。
```

### **8.3 事故车**

必显：

```text
事故车维修无法仅凭线上图片准确报价。
平台不提供线上最终报价，实际方案和费用需门店检测或拆检后确认。
你可以先查看类似脱敏案例，了解常见项目和费用影响因素。
```

禁止：线上最终价、按图报价、最低价/保证报价、固定维修金额。

## **9. 包含 / 不包含 / 适用场景**

### **9.1 包含示例**

指定配件、标准工时、基础检测；**门店支持时**可含维修过程服务相册记录。

### **9.2 不包含示例**

额外损坏、非套餐配件、特殊车型差价、拆检新增项目、到店后升级服务。

### **9.3 价格变化提示**

```text
如到店检测发现情况与所选服务不符，门店会与你确认新的方案和费用。
```

### **9.4 适用 / 不适用**

适用：列举典型症状/场景。不适用：引导至复杂维修说明或提交咨询/联系客服。

## **10. 价格影响因素**

通用：车型、配件品牌/规格、年款、损伤程度、拆装/喷漆、工时、新能源、门店检测结果。

事故车额外：结构件、气囊、底盘、电池包、定损、拆检、总成更换、二次损伤。

可折叠标题：「为什么价格不同」。

复杂度分级（L1–L4）用于标签或说明时见 `10_复杂度分级详细标准.md`（V2.0 仅作服务说明参考，不绑定订单/奖励规则）。

## **11. 服务流程（V2.0）**

### **11.1 标准/套餐服务**

```text
浏览服务说明 → 选择门店 → 提交咨询/预约到店
→ 门店电话联系 → 线下到店检测/施工 → 线下付款
→ 门店创建服务相册（若启用）→ 用户查看相册与配件确认
→ 可选：授权脱敏公开为案例
```

### **11.2 复杂维修**

```text
浏览价格说明与影响因素 → 选择门店
→ 填写问题描述、上传照片（可选）、期望到店时间
→ 提交咨询/预约 → 到店检测 → 线下确认方案与费用
```

### **11.3 事故车**

```text
阅读事故车说明 → 查看类似案例 → 选择门店
→ 填写损伤描述、上传照片（可选）
→ 勾选「知晓到店检测后报价」→ 提交预约
→ 到店检测/拆检 → 线下确认方案
```

**删除 V1.0 流程节点**：提交订单、微信支付、等待接单、确认完工、交易评价、平台售后。

## **15. 常见问题（V2.0）**

**Q：线上价格就是最终价格吗？**

标价为参考。套餐/一口价以页面说明为准；超出范围或复杂情况由门店线下确认。平台不提供线上最终报价担保。

**Q：事故车可以线上报价吗？**

不可以。需到店检测或拆检后由门店与你线下确认。

**Q：维修过程可以看到吗？**

支持辙见的门店可创建**服务相册**，你可在关联后于「我的服务相册」查看过程与配件确认（非订单进度页）。

**Q：不满意怎么办？**

与门店线下沟通。内容/隐私问题可联系平台客服；**平台不参与**维修质量仲裁、退款或售后赔偿。

**Q：如何预约？**

选择门店后点击「预约到店」或「提交咨询」，填写信息即转为咨询线索，门店将联系你。

## **20. 合规要求**

### **20.1 价格**

明确包含/不包含；区间展示影响因素；事故车无线上最终价；禁止最低价、保证修好、实际支付金额宣传。

### **20.2 案例**

仅脱敏、已审核、已授权/商家责任确认案例；无隐私字段。

### **20.3 分享**

分享标题内容型，如「杭州刹车片更换：流程与价格影响因素说明」。禁止分享有奖、浏览有奖、帮我赚钱等。

### **20.4 平台身份（详情页须可见）**

```text
平台提供服务项目说明、脱敏案例与服务相册工具。实际维修、报价、收款、发票、质保和售后由门店线下提供和承担。
```

## **25. P0 验收标准**

1. 可进入分类、列表、详情；
2. 价格模式与 PriceDisplay 四型一致；
3. 一口价/套餐展示包含与不包含；
4. 区间/事故车展示影响因素且无线上最终价；
5. 可查看可服务门店并进入门店详情；
6. 可选择门店并进入咨询/预约表单，预填服务与门店；
7. 底栏无「立即购买/提交订单/在线支付」；
8. 关联案例脱敏、标签为授权/商家历史；
9. 分享无诱导奖励；
10. 下架/不可预约状态正确拦截；
11. 未登录可浏览，提交咨询须登录+手机号。

### **1.1 平台身份（V2.0）**

平台**不是**：交易平台、维修服务提供方、统一报价方、维修质量担保方、售后赔偿主体。

平台**是**：**服务相册工具**、咨询预约与授权留痕工具、门店分享链路的 **私域承接端**；**GEO / 公开案例获客主战场在站外**（H5、公众号）。

统一说明（服务页、门店页、案例页、咨询页、相册页均需展示）：

> 平台提供门店信息展示、维修案例展示、服务相册、咨询预约和隐私脱敏工具。实际维修服务、检测报价、收款、发票、质保和售后由门店线下提供和承担。案例内容由门店上传或经用户授权后展示，仅供参考，不构成平台报价、维修承诺或质量保证。

## **2. 产品目标**

1. **看得懂**：看懂服务项目、价格影响因素、类似案例与维修过程。
2. **敢选择**：通过门店资质、脱敏案例、服务相册建立信任，降低不透明担忧。
3. **有档案**：线下维修后通过服务相册留存过程与配件确认记录；用户可授权脱敏公开。

V2.0 **不做**：在线下单、微信支付、订单履约、完工确认、交易评价奖励、售后仲裁。

## **3. 核心闭环**

### **3.1 小程序内（工具闭环 · 2026-06-08 起优先）**

```text
用户进入小程序（微信搜索 → 工具首页 / 商家分享 → 深链直达）
→ 扫码或输入相册码 / 打开「我的服务相册」
→ 查看服务相册六阶段记录
→ 可选：电话咨询 / 在线咨询（锁定来源门店）
→ 可选：授权公示（脱敏预览 → 留痕 → 平台审核 → 站外 H5/公众号/GEO 发布）
→ 可选：自主分享给好友（与授权公示无关）
```

### **3.2 站外（公域获客闭环 · H5 / 公众号）**

```text
授权案例审核通过 → H5 案例页 / 公众号文章 / GEO 收录
→ 站外浏览与搜索 → 咨询/电话 → 线索回流该门店
→ 到店服务 → 商家创建服务相册 → 回到 §3.1
```

### **3.3 历史 V2.0 发现闭环（仅深链兼容 · 待下线）**

原「首页搜索 → 浏览服务/案例/门店 Tab → 咨询」路径仍保留路由，**不再** 作为冷启动首页与主导航（见 `02_工具入口首页.md`、`08_双系统转型开发计划.md` 阶段 A/D）。

## **10. 案例、授权公示与分享**

### **10.1 授权公示**

- 案例来源：**用户授权案例**、**商家历史案例**（无「平台订单案例」）。
- **授权公示**：用户授权平台将脱敏内容发布到 **H5、公众号、站外 GEO** 等公开展示位（**不含** 小程序案例 Tab）；相册 **`completed` 即可发起**（含无过程图、缺关键节点）；**不得** 因留档不完整阻止车主提交；审核通过后 **自动发布至站外**（见 `00_Phase1_服务相册产品口径.md` §4.5、§4.5.6）。
- 平台公开展示 **强制脱敏 + 合规审核**（非维修质量/六阶段质检；见 `04_公开案例审核.md` §9.3）。

### **10.2 用户自主分享（私人传播）**

- 用户分享 **自己的** 服务相册/案例给好友、朋友圈、社交账号等，**不需要** 先完成授权公示。
- 分享前提供脱敏选项：**默认脱敏**，用户 **可选择原图分享**；平台展示风险提示并留痕，**不强制** 脱敏（见口径 §4.5.3）。
- **交互**：相册详情 **「分享」→ 半屏 ShareSheet**（`07_维修相册查看页.md` §17.4）。
  - **私人**：完工即可；**复制分享链接**（公网 `album/share.html?token=`）与 **转发好友** 均支持 **默认脱敏 / 可选原图**（token.mode，**不必授权公示**）。
  - **平台公示**（若已审核）：额外提供 **复制公示网页链接**（`case/view.html`，强制脱敏，与私人原图选项无关）。
  - 右上角「复制链接」仍为微信小程序链，**不能**替代上述公网 URL。
- 私人分享 **不** 自动进入 **H5 / 公众号等公示渠道**（须另行授权公示且审核通过）。

### **10.3 合规**

- V2.0 不做评价奖励、分享领钱类诱导。
- 禁止混用文案：「分享即公开」「须授权后才能分享」。

## **13. P0 验收标准**

### **13.1 工具化首页（2026-06-08 增补 · DS-A-02）**

1. 微信搜索冷启动见 **工具入口首页**（`02_工具入口首页.md` §14），无跨店推荐；
2. Tab 目标态为 **首页 + 我的**（服务/案例/门店 Tab 不暴露）；
3. 扫码 / 相册码 /（未登录）我的服务相册入口可用；**已登录相册摘要在「我的」**；
4. 提供 H5 内容站出口；分享深链仍可直达相册/门店。

### **13.2 通用能力（保留）**

1. 用户可微信登录、绑定手机号；
2. 可通过 **深链** 浏览服务/门店/案例详情并搜索（非冷启动主路径）；
3. 可查看服务/门店详情（含资质、案例、咨询入口）；
4. 可查看公开脱敏案例（**站外 H5 为主**；小程序详情页深链兼容）；
5. 可通过电话、在线咨询或表单提交咨询/预约，提交前确认线下服务条款；
6. 可查看咨询/预约记录；
7. 可通过链接/扫码查看私密服务相册；
8. 可在我的服务相册中查看已关联相册；
9. 可预览脱敏效果并同意/拒绝 **授权公示**；
10. 可在 **我的服务相册** 自主分享（默认脱敏、可选原图），且分享 **不依赖** 授权公示；
11. **站外** 可生成服务/案例/门店/车型/故障相关 GEO 页面（小程序 GEO Tab 非 P0）。

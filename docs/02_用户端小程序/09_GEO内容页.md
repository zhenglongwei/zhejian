## **2. 页面目标**

通过「城市 + 服务 / 车型 / 故障 / 门店 / 案例」组合，让用户找到门店与脱敏案例，并进入咨询入口。

> **2026-06-08 起**：上述目标在 **H5 / 公众号** 完成；小程序端仅 **深链兜底**。

---

## **3. 页面类型（P0 六类）**

| 类型 | 示例 |
| --- | --- |
| 城市服务页 | 杭州刹车片更换 |
| 城区服务页 | 杭州余杭区钣金喷漆 |
| 车型服务页 | 宝马 3 系刹车片更换案例 |
| 故障问题页 | 汽车刹车异响怎么办 |
| 门店 GEO 页 | 杭州某某汽车维修门店案例 |
| 案例聚合页 | 杭州事故车维修案例合集 |

**实现**：小程序 `pages/geo/detail` + H5（待 **DS-C-05** 专题 URL）；读 **`GET /user/geo-pages/:id`**。

## **4. 标准结构**

页面标题、一句话摘要、适用场景、价格影响因素、相关案例、相关门店、常见问题、咨询/预约入口、平台说明、更新时间。

**深链兼容期**：小程序页 **不得** 展示跨店「推荐 GEO / 更多专题」列表（单专题自闭环；见 **DS-A-09** 单店隔离精神）。

## **2. 页面目标**

通过「城市 + 服务 / 车型 / 故障 / 门店 / 案例」组合，让用户找到门店与脱敏案例，并进入咨询入口。

---

## **3. 页面类型（P0 六类）**

| 类型 | 示例 |
| --- | --- |
| 城市服务页 | 杭州刹车片更换 |
| 城区服务页 | 杭州余杭区钣金喷漆 |
| 车型服务页 | 宝马 3 系刹车片更换案例 |
| 故障问题页 | 汽车刹车异响怎么办 |
| 门店 GEO 页 | 杭州某某汽车维修门店案例 |
| 案例聚合页 | 杭州事故车维修案例合集 |

---

## **4. 标准结构**

页面标题、一句话摘要、适用场景、价格影响因素、相关案例、相关门店、常见问题、咨询/预约入口、平台说明、更新时间。

## **6. 合规说明（页脚）**

> 页面内容用于展示维修服务信息、门店信息和脱敏案例，不构成平台报价或维修承诺。实际维修方案、费用、配件、质保和售后由用户与门店线下确认。

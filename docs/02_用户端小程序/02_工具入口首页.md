## **2. 模块定位**

### **2.1 一句话**

小程序首页 = **服务相册工具入口**，不是平台发现页。

### **2.2 用户感知**

| 场景 | 用户应理解为 |
| --- | --- |
| 微信搜索首次进入 | 「这是一个查看门店为我创建的维修服务记录的工具」 |
| 商家分享相册/门店链接进入 | 「这是 XX 门店给我的相册/门店页」（单店上下文，见 DS-A-09～11） |
| 想找公开案例 / 比选门店 | 「应去 H5 内容站或公众号」，**不在** 小程序内完成 |

### **2.3 与旧首页的差异**

| 维度 | 旧 `02_首页与服务发现` | 本 PRD（工具入口） |
| --- | --- | --- |
| 定位 | GEO 引流 + 本地发现 | 相册查看 + 扫码/相册码入口 |
| 数据源 | `GET /user/home` 聚合推荐 | 本地最近访问 + 用户自己的相册列表 API |
| Tab | 五 Tab（首页/服务/案例/门店/我的） | **首页 + 我的**（可选「相册」快捷 Tab） |
| 公域出口 | 小程序内浏览案例/门店 | 引导至 **H5 内容站** |

本文档细化 `01_用户端小程序PRD V2.0.md` 中 **首页 Tab** 与 **冷启动入口** 的落地规格。

- **页面字段、模块顺序、交互与异常态**以本文档为落地依据；
- **授权公示渠道**以 Phase1 口径 §4.5 为准（H5 / 公众号 / 站外 GEO）。

服务/案例/门店/GEO **列表与搜索** 仍保留路由供 **深链兼容**，但 **不得** 从本首页或主导航导流（DS-DOC-03、DS-A-07）。

## **5. 首页整体结构**

```text
工具 Hero（主标题 / 副标题 / 工具说明，非促销 Banner）
主操作区
  - 扫描门店二维码（primary）
  - 输入相册码（secondary）
次操作区
  - 我的服务相册（未登录 → LoginSheet）
最近查看（localStorage 有记录时，仅 1 条）
我的服务记录（已登录 + 有关联相册时，列表为「自己的」，非推荐）
站外出口
  - 「想了解公开维修案例？前往辙见内容站」→ 复制 H5 链接（**条件展示**：公域搜索冷启动 + 无相册绑定 + 无商家扫码史，见 `utils/tool-entry-context.js`）
辅助入口
  - 我是商家 → 商家工作台
  - 联系客服 / 使用说明
底部 Tab：首页 | 我的
```

**模块优先级**（自上而下）：主操作 > 次操作 > 我的记录 > 站外出口 > 辅助入口。

## **6. 模块规格**

### **6.1 工具 Hero**

| 字段 | 规则 |
| --- | --- |
| 主标题 | **服务相册**（与 Phase1 口径一致；可用「维修服务记录」副语义，禁止旧品牌「维修档案/透明维修」） |
| 副标题 | 查看门店为你创建的维修服务记录 |
| 说明条 | 使用 `ComplianceNotice` 或等价短文案；**禁止**「平台推荐门店」「全网案例」类中介话术（设计体系 §2.4） |
| 禁止 | 促销 Banner、城市切换、搜索框跳转全站搜索、Hero 内嵌精选案例/门店 |

**推荐文案（P0）**：

- 身份说明（`HOME_PLATFORM_IDENTITY` 工具版，实现时写入 `constants/compliance-copy.js`）：  
  「辙见提供服务相册查看与咨询预约工具。实际维修、报价、收款与售后由门店线下提供和承担。」

### **6.2 主操作 · 扫描门店二维码**

| 项 | 规则 |
| --- | --- |
| 交互 | 点击 → `wx.scanCode` |
| 解析 | 支持门店码、相册邀请码、相册认领码（与 `packageMerchant/pages/album/invite`、`pages/album/claim` 现有链路对齐） |
| 成功 | 跳转对应相册详情 / 门店页 / 认领页 |
| 失败 | Toast「无法识别该二维码」；不 fallback 到首页推荐 |

实现任务：**DS-A-03**。

### **6.3 主操作 · 输入相册码**

| 项 | 规则 |
| --- | --- |
| 交互 | 点击 → 打开相册码输入（独立页 `pages/album/enter-code` **或** 首页 BottomSheet，二选一，P0 至少一种） |
| 校验 | 非空；格式错误友好提示 |
| 成功 | 跳转 `pages/album/detail`（或认领流程） |
| 禁止 | 输入框旁不出现「搜索全站案例/门店」 |

实现任务：**DS-A-04**。

### **6.4 次操作 · 我的服务相册**

| 状态 | 展示 |
| --- | --- |
| 未登录 | 按钮 + 说明「登录后查看门店为你创建的记录」→ `LoginSheet`（咨询语义） |
| 已登录 | 跳转 `pages/album/list` 或「我的 → 我的服务相册」同路由 |

### **6.5 最近查看（P1 · DS-A-05）**

| 项 | 规则 |
| --- | --- |
| 数据源 | `localStorage` / `wx.setStorage`；字段 `storeId`、`albumId`、`storeName`、`visitedAt` |
| 展示 | **仅 1 条**最近记录；文案示例：「最近查看：XX汽修 · 继续查看」 |
| 空态 | **不展示**该模块（不占位） |
| 禁止 | 展示多条、跨店推荐列表 |

### **6.6 我的服务记录（已登录）**

| 项 | 规则 |
| --- | --- |
| 数据源 | `GET /user/service-albums`（现有 API）；**仅当前用户关联**相册 |
| 展示 | 简版列表 3～5 条 +「查看全部」→ 我的服务相册 Tab 页 |
| 空态 | 「暂无服务相册；请扫描门店二维码或输入相册码」 |
| 禁止 | 混入其他用户或平台推荐相册 |

### **6.7 站外出口 · H5 内容站（P1 · DS-A-06）**

| 项 | 规则 |
| --- | --- |
| 文案 | 「想了解公开维修案例？前往辙见内容站」 |
| 行为 | 复制链接 `https://geo.simplewin.cn/`（或业务常量 `constants/h5-links.js`）+ 提示浏览器打开；**或** 企业主体允许时使用 `web-view` |
| 说明 | 公开案例、门店列表、GEO 专题在 **站外** 浏览；与 Phase1 §4.5 授权公示渠道一致 |

### **6.8 辅助入口**

| 入口 | 规则 |
| --- | --- |
| 我是商家 | → `packageMerchant/pages/workbench`（未入驻走 onboarding） |
| 联系客服 | 现有客服入口 / 电话（与 `12_我的页面` 一致） |
| 使用说明 | P1：静态说明页或 H5 帮助；文案区分 **车主工具路径** vs **商家工作台路径** |

## **7. 禁止模块（红线）**

以下模块 **不得** 出现在工具入口首页（设计体系 + 08 计划 §4.2）：

- 附近门店、精选案例、推荐服务、GEO 专题横滑
- 城市定位切换（用于发现）；`app.json` 定位权限诉求移除（**DS-A-08**）
- 跨店搜索入口（原 `ui-search-bar` 跳转 `pages/search`）
- 六宫格服务分类入口（原 `HOME_SERVICE_ENTRIES`）
- 事故车重点入口卡片（引导去 H5 或深链，不在冷启动首页主推）
- 任何「浏览全部门店 / 全部案例 / 查看更多门店」文案或按钮
- 调用 `GET /user/home` 渲染推荐模块

## **9. 页面状态**

| 状态 | 展示 | 说明 |
| --- | --- | --- |
| `loading` | `ui-skeleton` | 首屏仅等待「我的相册」接口（已登录时）；未登录可立即 `normal` |
| `error` | `ui-empty` + 重试 | 相册列表接口失败；主操作区仍可用 |
| `empty` | 无关联相册 | 展示扫码/相册码引导；**不** 展示推荐填充 |
| `normal` | §5 全模块 | 按登录态裁剪 §6.4～6.6 |

## **14. P0 验收标准**

1. 微信搜索冷启动（无分享参数）：见 **§5 工具首页**，**无** 门店/案例/GEO 推荐模块。  
2. 可扫码进入有效相册/门店链路。  
3. 可输入相册码进入相册详情（或明确错误提示）。  
4. 未登录可扫码；「我的服务相册」触发登录引导。  
5. 已登录且有关联相册：展示 **用户自己的** 简版列表，非推荐。  
6. Tab 仅 **首页 + 我的**（或含可选相册 Tab），**无** 服务/案例/门店 Tab。  
7. 提供 H5 内容站出口文案与跳转/复制行为。  
8. 「我是商家」可进入工作台/onboarding。  
9. 分享深链进入 **不** 被工具首页拦截。  
10. 文案通过 §2.4 自检（无中介平台话术）。

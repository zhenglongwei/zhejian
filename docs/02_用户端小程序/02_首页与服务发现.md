## **2. 模块定位**

> **历史 V2.0 定位（已 superseded）** — 下述「第一入口 / GEO 引流起点」仅适用于深链兼容期对照，**非** 2026-06-08 起产品战略。

首页与服务发现 **曾** 作为用户端第一入口，负责：

1. 让用户快速理解平台定位（案例 + 相册 + 本地门店，非交易平台）；
2. 引导用户找到需要的维修服务、门店与脱敏案例；
3. 通过真实案例、服务相册能力说明与资质门店建立信任；
4. 为**咨询/预约到店**、事故车检测预约、案例浏览提供入口；
5. 为服务详情、门店详情、案例详情、GEO 内容页导流；
6. 支持杭州本地服务发现；
7. 避免线上误导报价，尤其是事故车场景。

首页是 **GEO 引流 + 咨询线索** 链路的起点，不是交易链路起点。

## **4. 用户目标**

用户进入首页后典型任务：

1. 了解平台是否可信、做什么/不做什么；
2. 搜索或浏览需要的维修服务；
3. 查看类似脱敏案例与维修过程参考；
4. 查看附近或推荐门店；
5. 理解价格影响因素（非线上成交价）；
6. 判断应**电话咨询 / 提交咨询 / 预约到店**还是仅浏览案例；
7. 对事故车、钣喷、复杂维修建立正确预期；
8. 进入咨询、案例、门店或 GEO 专题流程。


## **8. 搜索框**

### **8.1 位置与 placeholder**

位于城市下方。默认：

```text
搜索服务、门店、车型、故障或案例
```

可轮播：刹车异响、钣金喷漆、事故车维修、宝马 3 系保养等。

### **8.2 搜索范围（P0）**

服务项目、门店名称、车型/故障关键词、公开案例标题与摘要、服务分类、GEO 页标题（若已收录）。

### **8.3 结果分类**

```text
服务 | 门店 | 案例 | GEO（P1）
```

车型/故障类搜索优先：**案例 → 相关服务 → 门店**。

### **8.4 结果卡片字段（摘要）**

**服务**：名称、分类、价格模式（一口价/参考区间/仅预约/事故车）、参考价（可选）、标签（标准/需检测/事故车）。

**门店**：名称、地址、距离（授权后）、资质标签、擅长项目、案例数、辙见标签（如「支持服务相册」）。**不展示**平台交易评分。

**案例**：标题、脱敏主图、服务项目、参考区间或「到店检测后确认」、门店（可选）、标签（**已脱敏 / 已审核 / 用户授权案例 / 商家历史案例**）。

### **8.5 空状态**

```text
暂未找到相关内容
你可以换个关键词试试，或查看热门服务
```

推荐：热门服务、事故车专题、精选案例、联系客服。

### **8.6 搜索合规**

不展示未审核案例、未脱敏图、隐私明文、未上架服务、冻结/下架门店；事故车结果不展示线上最终价；搜索日志不长期明文保留敏感信息。

详情见 `11_搜索与筛选.md`。

## **15. 辙见说明**

### **15.1 要点（3–4 条）**

1. **服务相册**：门店创建六阶段过程相册，用户可查看进度与配件确认；
2. **维修有档案**：线下服务过程可留存，便于复查与再次维修参考；
3. **公开先脱敏**：授权公开前隐藏车牌、人脸、VIN、手机号等；
4. **平台非交易方**：报价、施工、收款、质保由门店线下负责。

**删除 V1.0**：「平台订单生成档案」「订单中查看进度」「评价与申请售后」。

### **15.2 推荐文案**

```text
维修过程看得见
门店可记录检测、施工、完工等阶段，你可在服务相册中查看过程与配件确认。

每次维修有档案
线下维修过程可沉淀为个人服务相册，方便后续参考（需门店创建并关联）。

公开案例先脱敏
公开展示前会隐藏车牌、人脸、VIN、手机号等隐私信息。
```

## **19. 页面状态**

**首次加载请求**：城市、服务入口、推荐门店、精选案例、GEO 专题（若有）、平台说明、定位状态。

**部分失败**：服务入口可重试；门店/案例/GEO 模块失败则隐藏该模块；说明文案用默认兜底。

## **20. 合规要求**

1. 首页不出现诱导分享/浏览/点赞奖励文案；
2. **无**收益中心、领钱、好评返现类入口；
3. 事故车不承诺线上最终报价；
4. 案例/门店图须审核+脱敏；
5. 按钮文案使用「咨询」「预约到店」「联系门店」，禁用「立即购买」「提交订单」「在线支付」；
6. 分享卡片仅使用已脱敏、已授权/已审核的公开内容（分享规则见主 PRD §10）。

## **26. P0 验收标准**

### **26.1 首页基础**

打开即首页；默认杭州；拒绝定位可用；展示标语、服务入口、事故车入口、门店、案例、GEO（若启用）；五 Tab 正常；接口失败有兜底。

### **26.2 搜索**

可搜服务/门店/案例；无结果有空态；未审核/下架不出结果；无隐私明文。

### **26.3 服务与事故车**

入口可跳转；下架/即将开放不可进入咨询或支付；事故车无线上最终价。

### **26.4 门店与案例**

仅审核通过门店；案例脱敏主图；标签为授权/商家历史案例；无实际支付金额；点击进入详情。

### **26.5 合规**

无奖励诱导；无订单/支付主路径；平台身份说明可见；公开内容符合脱敏规则。

## **2. 模块定位**

「我的」是用户端个人服务中心，承载：

1. 登录与手机号绑定；
2. **我的咨询/预约**、**我的服务相册**、**我的公开授权**；
3. 收藏的服务/门店/案例（P1）；
4. 车辆信息（P1 可选；P0 可在咨询表单填车型）；
5. 帮助反馈、隐私设置、账号安全；
6. 商家工作台 / 平台运营工作台入口（按权限展示）。

## **5. 登录与手机号**

### **5.1 登录方式（P0）**

微信授权登录。

### **5.2 须登录的操作**

1. 提交咨询/预约；
2. 查看我的咨询记录；
3. 查看我的服务相册、配件待确认；
4. 授权/撤回公开案例；
5. 绑定车辆（若启用）；
6. 账号设置、注销申请。

### **5.3 可不登录浏览**

首页、服务、门店、案例、GEO 页、平台规则说明。

### **5.4 手机号绑定**

**用途**：咨询联系、相册关联、配件确认通知、账户安全。

**强制绑定场景**：提交咨询/预约；查看/认领服务相册；配件确认操作。

**方式（P0）**：微信手机号授权。

绑定前提示须符合平台身份说明（线下服务由门店负责）。

## **6. 我的咨询/预约**

详情见 `06_咨询记录与我的相册.md`、`05_咨询与到店预约.md`。

### **6.1 列表入口**

「我的咨询」→ 列表 Tab：全部 / 进行中 / 已关闭。

### **6.2 列表字段**

服务类型、门店、提交时间、状态（已提交 / 门店已查看 / 门店已联系 / 已取消 / 已关闭）、问题摘要。

### **6.3 详情页**

咨询状态、门店信息、问题描述与图片、期望到店时间、联系电话、「联系门店」「取消咨询」、平台说明。

底部固定说明：

> 本记录仅为咨询/预约信息，不代表线上交易订单。实际维修、报价与付款由你与门店线下完成。

### **6.4 不可出现**

待支付、待接单、维修中、待确认完工、退款、售后进度等订单态 UI。

## **10. 车辆管理（P1 可选）**

P0 可在咨询表单填写品牌/车系，不强制独立车辆模块。

若启用：

- 列表：品牌、车系、年款、脱敏车牌、默认车辆标记；
- 操作：新增、编辑、删除（逻辑删除）、设默认；
- 隐私：车牌脱敏；VIN/行驶证 P0 不采集；
- **删除提示**：删除后不再显示于列表，已关联的服务相册仍保留。

## **12. 设置页**

```text
账号与安全
手机号绑定
消息通知设置
隐私设置
清除缓存
用户协议 / 隐私政策
关于平台
退出登录
账号注销
```

### **12.1 退出登录**

提示：退出后将无法查看咨询记录、服务相册与授权状态，确定退出吗？

退出仅清除本地登录态，不删除服务端数据。

### **12.2 消息通知（P0 简版）**

可接收：咨询/预约相关、服务相册更新、**配件待确认**、案例审核结果、授权/下架、平台规则变更。

**不订阅**：订单状态、支付结果、评价、奖励到账、退款售后。

P1 可提供订阅消息开关说明页。

## **13. 隐私设置**

1. 是否允许案例相关通知；
2. 是否允许营销类通知（默认关，须明示同意）；
3. 授权公开记录与撤回入口跳转；
4. 不在公开场景展示完整手机号、车牌、VIN。

---

## **14. 账号注销**

### **14.1 前置条件**

1. 无进行中的隐私/投诉工单（平台定义）；
2. 无未处理的授权争议（可选校验）；
3. 用户确认不可恢复。

**删除 V1.0 条件**：无进行中订单、无待结算收益。

### **14.2 注销后**

1. 账号不可用；
2. 咨询记录、相册查看权限按法规与隐私政策保留或匿名化；
3. 已公开且已授权案例按规则处理（可下线或匿名化）；
4. 本地登录态清除。

提示文案须说明：平台不提供交易与资金托管，注销不涉及退款处理。

## **18. P0 验收标准**

1. 未登录可浏览公开内容，个人数据入口触发登录；
2. 支持微信登录与手机号绑定；
3. 我的首页展示：咨询、服务相册、公开授权、设置；
4. **不展示**订单、评价、奖励/收益入口；
5. 咨询列表与详情状态符合 V2.0 简化状态机；
6. 服务相册列表可进入详情与配件待确认；
7. 可查看授权状态并支持撤回（按规则）；
8. 帮助/客服含平台非交易声明；
9. 消息通知不含支付/订单/奖励类；
10. 商家/运营入口按权限展示；
11. 退出登录与注销流程可用且有合规提示。

## **2. 模块定位**

「我的」是用户端个人服务中心，承载：

1. 登录与手机号绑定；
2. **我的咨询/预约**、**我的服务相册**、**我的公开授权**；
3. 收藏公开案例与门店（P1，**仅供浏览参考**，非平台推荐）；
4. 车辆信息（P1 可选；P0 可在咨询表单填车型）；
5. 帮助反馈、隐私设置、账号安全；
6. 商家工作台 / 平台运营工作台入口（按权限展示）。

## **5. 登录与手机号**

### **5.1 登录方式（P0）**

微信授权登录。

### **5.2 须登录的操作**

1. 提交咨询/预约；
2. 查看我的咨询记录；
3. 查看我的服务相册、配件待确认；
4. 授权/撤回公开案例；
5. 绑定车辆（若启用）；
6. 账号设置、注销申请。

### **5.3 可不登录浏览**

首页、服务、门店、案例、GEO 页、平台规则说明。

### **5.4 手机号绑定**

**用途**：咨询联系、相册关联、配件确认通知、账户安全。

**强制绑定场景**：提交咨询/预约；查看/认领服务相册；配件确认操作。

**方式（P0）**：微信手机号授权。

绑定前提示须符合平台身份说明（线下服务由门店负责）。

## **8. 我的公开授权**

展示用户已授权或处理中的公开案例：

| 字段 | 说明 |
| --- | --- |
| 关联相册/案例 | 脱敏预览图 |
| 授权状态 | 待预览 / 已同意 / 已拒绝 / 已撤回 |
| 审核状态 | 待审核 / 已通过 / 已驳回 / 已下线 |
| 操作 | 查看服务相册、撤回授权（**含已公示案例**） |

**列表 Tag（Phase 1）**：**公开相册** / **审核中** / **私密相册**（与「我的服务相册」列表口径一致；此处不展示「已授权/已脱敏」）。

### **8.1 撤回授权规则**

用户在 **任何时候** 均可撤回已发起的公开案例授权，**不受审核状态限制**（含待审核、已审核通过并已公示）。

| 审核状态 | 是否可撤回 | 说明 |
| --- | --- | --- |
| 待审核 | ✅ | 审核队列中移除，案例不发布 |
| 已通过 / 已公示 | ✅ | 公开案例立即下线（H5 noindex / 404，移出 sitemap） |
| 已驳回 | — | 未公开，无需撤回 |
| 已撤回 | — | 已从列表移除或展示「已撤回」 |

撤回后：私密服务相册仍保留；已产生的传播数据保留后台记录；搜索引擎缓存清除存在延迟（见 `10_隐私协议与授权规则.md` §10.2）。

规则详见 `docs/04_维修过程相册/03_服务相册流程.md` §7.5 与 `10_隐私协议与授权规则.md` §10。

提示：退出后将无法查看咨询记录、服务相册与授权状态，确定退出吗？

退出仅清除本地登录态，不删除服务端数据。

### **12.2 消息通知（P0 简版）**

可接收：咨询/预约相关、服务相册更新、**配件待确认**、案例审核结果、授权/下架、平台规则变更。

## **13. 隐私设置**

1. 是否允许案例相关通知；
2. 是否允许营销类通知（默认关，须明示同意）；
3. 授权公开记录与撤回入口跳转；
4. 不在公开场景展示完整手机号、车牌、VIN。

---

## **14. 账号注销**

### **14.1 前置条件**

1. 无进行中的隐私/投诉工单（平台定义）；
2. 无未处理的授权争议（可选校验）；
3. 用户确认不可恢复。

**删除 V1.0 条件**：无进行中订单、无待结算收益。

### **14.2 注销后**

1. 账号不可用；
2. 咨询记录、相册查看权限按法规与隐私政策保留或匿名化；
3. 已公开且已授权案例按规则处理（可下线或匿名化）；
4. 本地登录态清除。

提示文案须说明：平台不提供交易与资金托管，注销不涉及退款处理。

## **18. P0 验收标准**

1. 未登录可浏览公开内容，个人数据入口触发登录（`tool-login-copy` 统一文案）；
2. 支持微信登录与手机号绑定；
3. 我的首页展示：**服务记录摘要**、咨询、服务相册、公开授权、设置；
4. **不展示**订单、评价、奖励/收益入口；
5. 咨询列表与详情状态符合 V2.0 简化状态机；
6. 服务相册摘要 + 列表可进入详情；
7. 可查看授权状态并支持撤回（**含已审核通过并已公示的案例**）；
8. 联系客服 **非占位**（Modal + 联系方式 ActionSheet）；帮助入口合并；
9. 消息通知不含支付/订单/奖励类；
10. 商家入口在「我的」，**不在工具首页重复**；
11. 退出登录与注销流程可用且有合规提示；
12. 页脚合规说明 **仅「我的」Tab** 展示。

## 2. 模块定位

我的页面与账户体系是用户端的个人服务中心。

核心目标：

1. 承载用户登录与授权；
2. 展示订单、预约、评价、相册、档案等个人数据入口；
3. 支持用户管理车辆信息；
4. 支持手机号绑定和账号安全；
5. 支持奖励、收益、优惠等资产查看；
6. 支持用户联系客服、查看规则、设置隐私；
7. 为后续会员体系和复购运营提供基础。

2. 无未完成售后；
3. 无未完成退款；
4. 无待结算收益；
5. 无提现处理中记录；
6. 无平台风控限制；
7. 无未履行的平台规则义务；
8. 符合法律法规和平台数据保留要求。

### **21.4 不允许注销提示**

示例：

```
当前账号存在未完成订单，暂无法注销。

```

```
当前账号存在待处理售后，暂无法注销。

```

```
当前账号存在待结算收益，请处理完成后再申请注销。

```

### **21.5 注销流程**

```
用户进入账号注销页
→ 查看注销说明
→ 勾选确认
→ 身份校验
→ 系统校验注销条件
→ 提交注销申请
→ 平台处理
→ 注销完成

```

### **21.6 注销后处理**

注销完成后：

1. 用户无法登录原账号；
2. 登录态失效；
3. 用户个人资料按法规和平台规则处理；
4. 历史订单按法规要求保留必要记录；
5. 已公开案例需按授权撤回或脱敏保留规则处理；
6. 已展示评价可匿名化或下架，按平台规则；
7. 奖励、优惠券、收益失效，除非法律另有要求。

## **13. 公开授权入口**

### **13.1 入口定位**

订单完成后，可在服务相册或服务相册页展示：

```

```

### **13.2 展示条件**

必须满足：

1. 订单已完成；
2. 相册图片数量达到最低要求；
3. 无未处理售后争议；

点击授权前展示：

```
你的服务相册可供其他车主参考。公开并通过平台审核后，可按平台规则参与利益共享（以政策为准）。平台会自动脱敏车牌、手机号等信息，请你核对后再确认公开。公开版不展示完整订单信息与实际支付金额。
```

确认后记录 `user_rejected` 状态；相册仍可正常查看，后续可在规则允许时重新发起公开（P1）。

1. 若商家已提交完工且 **`order_pre_mask` 任务** `preMaskStatus=ready`，直接进入 `pages/desensitize/preview` 预览工作台，**无需等待** AI 脱敏；

### **14.4 档案字段展示映射（阶段 E · UI-ALB-E-01）**

> **口径**：下列字段对标行业电子维修档案的 **13 项子集**，在 Phase 1 **仅作展示聚合**，不要求商家按监管表单逐项上传；数据来自现有相册字段与六阶段节点，由 **B-ALB-08** 服务端聚合。

#### 列表 `summaryLine` 拼接规则（优先级从左到右，最多 3 段）

| 段 | 来源字段 | 聚合规则 | 缺省 |
| --- | --- | --- | --- |
| 故障/损伤 | `issueDesc` | 接车节点 `note` 首句；或 `vehicle.issueDesc`；或检测节点 `note` 摘要（≤24 字） | 省略该段 |
| 配件摘要 | `partsSummary` | `partsJson` 前 2 项：`名称×数量`；超出加「等 N 项」 | 省略该段 |
| 参考价 | `referencePriceText` | 有 `planAmount` → `参考¥{planAmount}`（PriceDisplay **fixed** 口径）；否则区间价文案 | 省略该段 |

示例：`制动异响 · 刹车片×2 · 参考¥1280`（段间 ` · `，整行超 40 字尾部 `…`）。

#### 抽屉 `KeyInfoTable` 行（`summaryRows` 扩展）

| 标签 | 来源 | 节点/字段映射 |
| --- | --- | --- |
| 送修日期 | `deliverDateText` | `createdAt` 格式 `YYYY-MM-DD` |
| 更新时间 | `updatedAtText` | 列表已有 |
| 服务项目 | `serviceName` | 相册顶字段 |
| 门店 | `store.name` | 相册顶字段 |
| 车辆 | `vehicleDisplay` | `vehicleJson` 格式化 |
| 里程 | `mileageText` | `vehicleJson.mileage` + `km`；无则「—」 |
| 故障/损伤 | `issueDesc` | 接车 `stage_1.note` 或 `vehicle.issueDesc` |
| 检测结果 | `inspectionResult` | 检测 `stage_2.note` 摘要 |
| 维修方案 | `repairSolution` | 方案节点 `stage_3.note` 或 `storeNote` 摘要 |
| 配件摘要 | `partsSummary` | 同列表段 |

## **16. 与「发布到公开网站」的关系（V2.0 · 修订 2026-07-18）**

服务相册本身不绑定交易评价或现金奖励。

**发布到公开网站**（非「发给微信 / 复制去社交媒体」）的条件：

1. 车主主动点击 **发布到公开网站**（预览 = 即将上网内容）；
2. 公开素材 **强制脱敏**（PublicView）；
3. **人工审核**通过；
4. **未点发布则不上** H5 / GEO。

通过后展示于 **H5 公开案例页 / 站外 GEO**（**不含** 小程序案例 Tab）。历史用语「授权公示」= 本路径（见口径 §4.5）。

V2.0 **不做**：订单完成评价、评价奖励、分享领钱。可做 **荣誉/心理鼓励**（无现金）。

**口径**：`00_Phase1_服务相册产品口径.md` §4 / §4.5（用户主导发布）。

## **17. 分享规则**

> **定稿 2026-07-18（卷十二 USER-PUB）**：**发给微信 / 复制去社交媒体** 与 **发布到公开网站** 分离。前两者 **不要求** 先上公开网站；公开网站须 **车主点一下 + 人工审核**。私人与社交媒体链路 **仅脱敏图/文**（原图分享已废止，见口径 §4.5.4）。

### **17.1 发给微信与复制社交媒体（不上公开网站）**

| 项 | 规则 |
| --- | --- |
| **谁可以** | 关联车主 |
| **何时** | 相册 **已完工**；**不必** 先发布到公开网站或过审 |
| **发给微信** | 好友、朋友圈（原生能力）+ 复制私人分享链接 |
| **复制去社交媒体** | 小红书 / 知乎 / 头条等长文 + **一键复制**；车主自行粘贴；**商家/平台不代发** |
| **脱敏** | **强制仅脱敏**；完整原图仅小程序内登录态可见 |
| **与公开网站** | **不** 自动进入 H5 / GEO |

私密服务相册 **不可** 把「仅登录可见的相册详情链接」当公开外链；须走分享能力生成的脱敏链接/文案。

### **17.2 发布到公开网站**

车主在 ShareSheet 选择 **发布到公开网站** → 预览 PublicView → 确认留痕 → `pending_review` → **人工审核通过** → H5 / GEO。

此路径 **强制脱敏 + 审核**；与是否发给微信、是否复制到小红书 **无关**。  
**禁止** 未确认即上网；**禁止**「分享即公开」。

### **17.3 分享文案限制与荣誉鼓励**

分享卡片 / 按钮 **不得**出现：

```
分享有奖
点击有奖
点赞有奖
浏览有奖
帮我赚钱
转发领钱
```

可使用内容型 / 荣誉向文案：

```
这次维修过程已整理，可发给朋友参考。
帮助同城车主少踩坑（发布到公开网站后）
```

后台可做传播归因；**无** 用户侧现金奖励结算。物质感谢由门店线下自行处理，平台不承诺、不结算。

```

```

### **20.4 订单售后中**

如订单存在售后争议，公开授权入口可隐藏或置灰：

```
订单售后处理中，暂不可公开为案例。

```

### **20.5 用户撤回授权**

P0：用户在 **任何时候** 均可撤回公开授权（含案例已审核通过并已公示）。

撤回后：

1. 公开案例下架；
2. 已产生的传播数据保留后台记录；
3. 奖励是否撤销按奖励规则处理；
4. 私密服务相册仍保留。

## **21. 合规要求**

1. 原始服务相册默认私密；
2. 未经用户授权不得公开；
3. 公开前必须脱敏；
4. 公开前必须审核；
5. 不得展示完整车牌、VIN、手机号、证件；
6. 不得展示用户实际支付金额；
7. 不得把私密相册链接分享给陌生用户；
8. 不得在分享入口展示诱导奖励文案；
9. 事故车案例公开时不得暗示线上最终报价；
10. 涉及保险、事故责任、违法信息时需谨慎审核。

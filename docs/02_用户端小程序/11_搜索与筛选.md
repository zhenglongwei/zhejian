## **2. 模块定位**

> **历史 V2.0 定位（待下线）** — 「缩短需求 → 案例/服务/门店 → 咨询」的 **小程序内全站搜索** 由 **H5 内容站** 承接。

帮助用户快速找到可浏览、可咨询的内容（**曾** 为小程序内目标），缩短 **需求 → 案例/服务/门店 → 咨询预约** 路径，并为运营与 GEO 提供需求数据。

核心能力：

1. 关键词搜索服务、门店、公开案例；
2. 搜索 GEO 专题页（P0 简版 / P1 完整 Tab）；
3. 按距离、价格模式、资质、相册能力等筛选；
4. 按服务类型、车型、门店能力过滤；
5. 隐私与合规控制（不泄露私密相册与敏感信息）。

首页搜索入口摘要见 `02_首页与服务发现.md` §8；本文档为搜索全流程细则。

## **5. 搜索对象**

### **5.1 可搜索（公开）**

| 对象 | 说明 |
| --- | --- |
| 服务 | 小保养、轮胎、钣喷、空调等 |
| 门店 | 名称、地址、服务能力 |
| 公开案例 | 已审核、已脱敏 |
| 服务分类 | 保养、维修、钣喷等 |
| 品牌/车型 | 宝马 3 系、Model Y |
| 症状/标签 | 异响、剐蹭、补漆 |
| GEO 页 | 城市/车型/故障专题标题（P0 可合并进「全部」） |

### **5.2 不可搜索（公开索引）**

1. 用户私密**服务相册**（未授权公开部分）；
2. 未脱敏/未审核案例；
3. 用户手机号、车牌、VIN 明文；
4. 咨询记录详情（非公开线索）；
5. 后台内部备注、投诉工单原文。

**删除 V1.0**：用户私密订单、未审核评价、售后争议正文作为可搜对象。

## **6. 搜索页结构**

```text
搜索输入框（自动聚焦、清空、回车搜索）
取消 → 返回上一页
搜索历史（最多 10 条，P0 支持清空全部）
热门搜索
推荐服务 / 门店 / 案例（可选）
搜索联想（输入 ≥1–2 字，P0 可简版）
```

P1：语音输入、单条删除历史、登录用户云端同步历史。

## **7. 热门搜索与联想**

### **7.1 热词来源**

运营配置、搜索热度、城市热门服务、季节服务、案例高频标签。

示例：小保养、补漆、轮胎更换、电瓶更换、空调不制冷、事故车检测。

**热词禁止**：绝对化价格、事故车最终价、诱导分享、虚假品牌词。

### **7.2 联想类型**

服务、症状、案例、门店、品牌车型。展示：关键词 + 类型标签 + 结果数（可选）。

### **7.3 同义词（P1 配置）**

喷漆=补漆=漆面修复；保养=小保养；电瓶=蓄电池。

## **13. 筛选**

### **13.1 通用**

| 筛选项 | 说明 |
| --- | --- |
| 城市/区域 | MVP 默认杭州 |
| 距离 | 1/3/5/10 km（需定位） |
| 服务分类 | |
| 价格区间 | 仅对有标价的标准服务 |
| 营业中 | |
| 可咨询/预约 | |
| 支持服务相册 | |
| 新能源/事故车能力 | |

**删除 V1.0**：评分、是否支持线上支付。

### **13.2 服务筛选**

分类、价格区间、价格模式、可服务门店、适用车型、是否套餐、是否需到店检测。

### **13.3 门店筛选**

区域、距离、资质、营业状态、服务能力、品牌（有授权时）、新能源、事故车、服务相册、可预约时段（P1）。

### **13.4 案例筛选**

服务类型、品牌/车型、维修类型、是否有相册/前后对比、门店、发布时间、案例来源（授权/商家历史）。

## **14. 排序**

| 排序 | 适用 |
| --- | --- |
| 综合（默认） | 全部 |
| 距离最近 | 门店 |
| 价格从低到高/高到低 | 有明确标价的服务 |
| 最新发布 / 热度 | 案例 |
| 案例数优先 | 门店（可选） |

**删除 V1.0**：评分最高、销量优先、按订单量。

**综合排序因素**：关键词匹配、城市、距离、上架/营业状态、案例质量、相册能力、可咨询状态、运营权重。**不含**交易评分与订单量。

P1 商业推广结果须标识「推广」；P0 不混入广告排序。

---

## **15. 定位与城市**

授权提示：「开启定位后，可为你推荐附近门店和服务。」

未授权：用手动/默认城市（杭州），可搜服务与案例，距离排序置灰。

切换城市后刷新结果与热词（P1 多城市）。

---

## **16. 空结果**

**无关键词命中**：

```text
暂无相关结果，换个关键词试试。
```

推荐：热门服务、推荐门店、相关案例、联系客服（P1）。

**筛选过窄**：「当前筛选条件下暂无结果」+ 清空筛选。

**城市无内容**：「当前城市暂未开通相关服务，可查看其他内容或切换城市。」

---

## **17. 敏感词与纠错**

敏感/违规词：不返回违规内容，可提示无结果，记风控日志。

P1：错别字纠错（「你是否想搜索：补漆」）。

用户搜索车牌/手机号/VIN：**不做公开联想**，返回「未找到相关公开内容」。

---

## **18. 合规**

1. 不泄露隐私；私密服务相册不可被搜；
2. 事故车结果无线上最终价；
3. 案例须脱敏审核；
4. 热词无虚假承诺与分享诱导；
5. **不展示、不排序**交易评价与虚假销量；
6. 定位遵守微信与隐私政策；
7. 搜索日志不长期明文存敏感词。

---

## **19. 异常**

网络/搜索服务/分页失败/关键词过长（建议 ≤30 字）均有明确提示与重试。

## **21. 接口建议（摘要）**

```text
GET  /api/user/search/config
GET  /api/user/search/suggest?keyword=
GET  /api/user/search?keyword=&type=&city_code=&sort=&page=
GET  /api/user/search/services | /merchants | /cases | /geo-pages
GET  /api/user/search/hotwords
POST /api/user/search/history
DELETE /api/user/search/history
```

请求体示例字段：`keyword`、`city_code`、`tab`（service/merchant/case/geo/all）、`filters`（含 `support_album`、`distance`、`category_id` 等）、`sort`、`page`、`page_size`。

**服务结果** 含 `price_mode`、`price_display`、`tags`、`available_merchant_count`；**不含** `rating`、`sales_count`。

**门店结果** 含 `distance`、`business_status`、`tags`；**不含** `rating`。

**案例结果** 含 `title`、`cover_url`、`vehicle_display`、`service_type`、`source_tag`（authorized/merchant_history）。

1. 可从首页进入搜索并提交关键词；

7. 案例脱敏、来源标签正确；
8. 事故车相关无线上最终价；
9. 未授权定位仍可搜索；

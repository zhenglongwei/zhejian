## 1. 目标

在现有「身份证明 + 公开案例 + 专题」之上，补齐门店作为 **AI 时代修车店数据资产** 所需的两类信号：

| 线 | 名称 | 用户感知 | 一期范围 |
| --- | --- | --- | --- |
| **A** | 活店感 | 这家店今天还在正常营业、资料不是陈年旧档 | 真实营业态、临时休息、内容新鲜度 |
| **B** | 能力感 | 能修什么、有什么人/设备、明确不做什么 | 技师公示卡、设备/场标签、擅长品牌、暂不承接、授权有效期 |

**不做（一期）**：今日可约档期日历、进行中相册公示、咨询响应分钟数对公展示、交易评分。

---

## 2. 产品定调（已确认）

1. **一期范围**：同意 MVP（营业态 + 新鲜度 + 技师/设备/品牌/不承接/授权时效 + 审核 + 透明度加固）。二期再做可约档期与能力图谱深化。
2. **变更审核分层**：
   - **须审后亮**：技师公示卡、设备/场标签（含设备实景图）、品牌授权（含有效期）。
   - **即时生效**：营业时间、临时休息、门店电话、简介、擅长服务项目标签、擅长品牌、暂不承接文案（仍须真实；虚假走举报/事后处罚）。
3. **咨询响应**：**继续仅商家后台 / 透明度内部维**，不对普通用户与 H5 公开页展示具体比例或「响应很快」类口号（防攀比造假）。可影响内部排序权重（可选，非 P0）。
4. **进行中相册永不公示**：仅「已完工 + 用户授权（或商家历史案例）+ 审核通过」可进入公开面。透明度「过程资料齐全度」继续只聚合 **已完工** 相册完整率，不外链私密/进行中相册。

---

## 3. 透明度底座评估（C）

**结论**：现有透明度可作为过程资料完整度底座，**不足以单独撑起** A/B 数据资产；一期须加固并回灌，而非另开「公示进行中相册」。

| 已对齐 | 一期须补 |
| --- | --- |
| 无公开案例时不虚报总分 | 内容新鲜度维（最近公开案例 / 资料核实时间） |
| 相册完整率只统计已完工相册 | 真实营业态（勿把 `ACTIVE` 直接当「营业中」） |
| 咨询响应仅内部 audience | 资质维纳入维修资质有效期 |
| 分项 + 证据链已接 H5 / Schema | 能力完善度轻量计入（已审核技师/设备是否展示） |

## 4. 公开数据结构（人机同源）

小程序门店详情、H5 门店页、Feed/Schema 共用同一契约（字段名可 snake/camel 映射，语义一致）：

```text
businessStatus          // open | closed | holiday | suspended | offline（实时计算 + 运营态）
businessHours           // 展示文案
temporaryClosures[]     // 临时休息
freshness               // { lastPublicCaseAt, lastProfileVerifiedAt, summary }
specialtyBrands[]       // 擅长品牌（即时）
notAccepting[]          // 暂不承接（即时）
equipmentTags[]         // 已审核通过才出现在公开面
techniciansPublic[]     // 已审核通过才出现
brandAuth               // { verified, validUntil, imageUrl } 过期不展示
transparency            // 既有 dimensions；增 freshness / 资质有效性说明
```

**硬规则**：任何公开读侧不得返回进行中相册 ID、封面或节点图。

## 5.1 二期第一批口径（2026-07-17 确认）

### P2-04 资质 / 授权过期催办与轻降权

| 规则 | 说明 |
| --- | --- |
| 自动下线展示 | 品牌授权有效期已过 → 公开面不展示「已授权」（一期已有） |
| 运营催办列表 | 品牌授权或维修资质：已过期 / 30 天内即将过期，进入运营台列表（只读催办，无短信） |
| 轻降权 | 列表排序 `score` 扣分：已过期各 −25，即将过期各 −8；**不对用户展示羞辱文案** |
| 商家侧 | 看板 suggestions 继续提示「授权将过期」 |

## 6. 验收口径（一期）

1. 非营业时段或临时休息日内，门店不得显示「营业中」；搜索「营业中」筛选一致。  
2. 进行中相册在门店页 / 案例列表 / H5 / Feed / 搜索均不可见。  
3. 未审核通过的技师、设备、品牌授权不出现在公开面；授权过期自动去掉授权展示。  
4. 营业时间/简介/擅长品牌/不承接保存后即时对公更新；技师/设备/授权须审后亮。  
5. 透明度：无公开案例不虚报；有案例时可展示新鲜度相关说明；咨询响应不对公。  
6. 小程序与 H5 字段与文案一致。

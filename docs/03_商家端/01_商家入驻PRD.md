## **2. 入驻目标**

1. **确认商家是谁**：营业执照、主体、负责人。
2. **确认门店在哪**：名称、地址、地图坐标、门头/工位照片。
3. **确认能做什么**：维修资质、擅长项目、新能源/事故车能力。
4. **复杂度与资质匹配**：L1–L4 可展示/承接范围提示（见 `10_复杂度分级详细标准.md`）。
5. **沉淀内容资产**：门店主页、服务方案、历史案例、服务相册能力。

**删除 V1.0**：用户评价、订单履约、接单权限作为入驻产出。

---

## **3. 角色**

### **3.1 商家主账号**

创建主体与门店、上传资质、提交审核、管理员工、创建服务方案与案例/相册。

### **3.2 平台审核人员**

审核主体、门店、资质、照片、服务能力真实性；通过/驳回/要求修改。

### **3.3 平台超级管理员**

修改商家状态、冻结/关闭门店、调整服务展示权限、分配运营。**V2.0 MVP 无**交易佣金配置。

---

## **4. 入驻入口**

```text
我的 → 商家工作台 → 申请入驻
```

未绑定手机号须先绑定。

### **4.1 未入驻页**

```text
成为辙见服务商
把每一次维修过程变成可信案例，让专业被用户看见
```

价值点：本地咨询线索、脱敏案例展示、门店 GEO/主页、服务相册透明化。

按钮：立即入驻、查看入驻要求、联系平台。

**删除**：「把每一单维修变成案例」「前期免佣」等交易导向表述（可改为「内容展示与咨询转接工具」）。

---

## **5. 入驻流程**

```text
绑定手机号
→ 填写商家主体信息 + 营业执照
→ 填写负责人信息
→ 创建门店（地址、地图、照片）
→ 上传维修资质
→ 选择擅长服务
→ 提交审核
→ 平台审核
→ 审核通过 → 商家工作台开通
```

多门店：主体通过后 → 门店管理 → 添加门店 → 单独审核 → 上线。

## **8. 维修资质**

### **8.1 必传**

营业执照、机动车维修经营资质/备案、门头、工位、负责人、地址、服务能力说明。

品牌授权：非必填；审核通过后可展示。

### **8.2 资质字段**

维修资质类型、资质照片、有效期、编号；承接新能源须新能源专项资质。

### **8.3 资质与 L 级（展示/创建约束）**

| 资质 | 可展示/创建范围（摘要） |
| --- | --- |
| 三类及以上 | L1、部分专项 |
| 二类/三类专项 | L2 维保、轮胎、电瓶等 |
| 二类及以上 | L3 中复杂项目 |
| 一类 | L4 事故车、大修等 |
| 新能源专项 | 三电相关（最低 L3，大修 L4） |

超出资质选择服务时：可存草稿，**不可提交审核**；提示补充资质。

系统根据资质**提示**可发布服务范围，**不做**平台派单过滤。

---

## **9. 擅长服务**

小保养、刹车、电瓶、轮胎、空调、钣喷、底盘、事故车、新能源等。须与资质一致，审核时校验是否夸大。

## **10. 入驻审核**

| 状态 | 说明 |
| --- | --- |
| 草稿 | 填写中 |
| 待审核 | 已提交 |
| 审核通过 | 工作台可用 |
| 审核驳回 | |
| 要求修改 | |
| 已冻结 | 限制经营 |
| 已关闭 | 不再展示 |

审核项：执照真实性、地址与定位、门头工位真实、资质匹配、擅长项目、新能源/品牌授权凭证。

**通过后可用**：服务方案、历史案例、**服务相册**、咨询线索、员工管理、门店主页。**不可**：平台订单接单（V2.0 无）。

## **11. 员工管理**

添加入口：工作台 → 员工管理 → 手机号邀请 → 选门店与权限。

| 权限 | 说明 |
| --- | --- |
| 查看咨询线索 | 本门店线索 |
| 处理咨询线索 | 标记已联系/关闭 |
| 服务相册 | 创建/上传/配件确认 |
| 历史案例 | 创建草稿/提交审核 |
| 服务方案 | 创建/编辑（按角色） |
| 门店资料 | 默认不可改 |
| 员工管理 | 主账号 |

**删除 V1.0**：查看订单、接单/拒单、提交完工。

## **12. 工作台首页（审核通过后）**

与 `00_商家工作台PRD V2.0.md` 一致：

待办：咨询线索、待车主确认相册、待补节点相册、待审核案例。  
快捷：新建服务相册、历史案例、服务方案、门店主页预览。  
简版数据：今日线索、新增相册、公开案例数、浏览量（P1）。

**删除**：今日订单、待接单、维修中、待完工。

---

## **13. 门店主页**

审核通过后生成：名称、地址、导航、营业时间、电话、资质、擅长、照片、服务方案、公开案例、辙见/服务相册说明。

支持小程序、H5、分享、GEO/SEO 基础结构。

**不展示**：平台交易评价区（V2.0）。

---

## **14. 合规提示（提交前）**

```text
请确认你提交的商家主体、门店地址、维修资质和服务能力真实有效。
平台会对资料进行审核，并可能要求补充证明。
如存在虚假资料、冒用资质、夸大服务能力或侵犯客户隐私等行为，平台有权下架门店、限制展示或终止合作。
```

---

## **15. P0 验收标准**

1. 可申请入驻并完成主体/门店/资质填写；
2. 地图选点、门头工位上传；
3. 擅长服务与资质校验；
4. 平台审核通过/驳回/要求修改；
5. 通过后进入 V2.0 工作台（无订单模块）；
6. 多门店与员工邀请；
7. 品牌授权可选；
8. 员工权限不含接单/订单。

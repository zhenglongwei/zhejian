## 4. 支付

- 渠道：微信小程序 **JSAPI 微信支付**（商户平台已配置）
- 流程：创建订单 → prepay → `wx.requestPayment` → 回调激活套餐
- 开发环境：未配置微信支付时，prepay 返回 mock 提示，可走 `mock-pay` 联调
- 样板店：运营标记 `founderFlag` + 赠送深度版；续费 `founderRenewDiscount=0.5`

### 环境变量（backend）

| 变量 | 说明 |
| --- | --- |
| `WECHAT_PAY_MCH_ID` | 商户号 |
| `WECHAT_PAY_API_V3_KEY` | APIv3 密钥 |
| `WECHAT_PAY_CERT_SERIAL` | 商户证书序列号 |
| `WECHAT_PAY_PRIVATE_KEY` 或 `WECHAT_PAY_PRIVATE_KEY_PATH` | 商户私钥 **apiclient_key.pem**（与证书同次下载） |
| `WECHAT_PAY_NOTIFY_URL` | 可选，默认 `{PUBLIC_BASE_URL}/api/v1/pay/wechat/notify` |

**验签失败（SIGN_ERROR）排查**：

```bash
cd backend && node scripts/wechat-pay-smoke.js
```

1. `WECHAT_PAY_CERT_SERIAL` = `openssl x509 -in apiclient_cert.pem -noout -serial` 输出去掉 `serial=` 与冒号，**大写**  
2. `WECHAT_PAY_PRIVATE_KEY` = 同目录 **apiclient_key.pem**（勿用 APIv3 密钥、勿用公钥）  
3. `WECHAT_PAY_MCH_ID` = 商户平台商户号，与证书一致  
4. 小程序 **AppID** 已在商户平台绑定为 JSAPI 支付 AppID  
5. 改 `.env` 后：`pm2 restart zhejian-api --update-env`

## 5. 车主评价 × 相册反馈（Phase 2 · 已落地）

用户确认：**评价**针对本次维修服务/商家；**反馈**针对相册功能或 UI。二者已整合入口：

| 类型 | 对象 | 入口 | 可见性 |
| --- | --- | --- | --- |
| 服务评价 | 相册对应的一次服务 | 相册尾页「评价与反馈」→ 服务评价 | 车主可选授权公开摘要；商家可回复；挂载于已审核公开案例 |
| 问题反馈 | 相册产品体验 | 同上 → 相册问题反馈（原 `album/feedback`） | 转达门店 + 产品改进，非星级评价 |

**不做**：V1 订单绑定六维交易评价、好评返现。

### 5.1 数据与 API

表：`service_album_reviews`（一相册一车主一条评价）

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| GET | `/user/service-albums/:albumId/review` | 评价资格与已提交记录 |
| POST | `/user/service-albums/:albumId/review` | 提交评价 |
| GET | `/merchant/reviews` | 列表 `tab=pending\|replied\|all` |
| GET | `/merchant/reviews/stats` | 待回复数 |
| GET | `/merchant/reviews/:id` | 详情 |
| POST | `/merchant/reviews/:id/reply` | 商家回复 |

公开案例 `getCaseDetail` 返回 `ownerReviews`（`authorizePublic=true` 且脱敏展示）。

### 5.2 小程序页面

| 端 | 路径 |
| --- | --- |
| 车主 | `pages/album/engage/index` · `pages/album/review/submit` · `pages/album/review/result` |
| 车主 | `pages/album/feedback/*`（保留） |
| 商家 | `packageMerchant/pages/review/list` · `detail` |
| 公开展示 | `pages/case/detail` · 车主服务评价区块 |

**完工门槛**：相册 `status=completed` 方可提交服务评价。

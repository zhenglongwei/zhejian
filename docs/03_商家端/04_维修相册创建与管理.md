## **2. 模块定位**

服务相册是平台 **P0 核心能力**，Phase 1 定位 **留档为主**：

1. 记录维修六阶段过程、配件类型、参考价与凭证图（**争议证据**）；
2. 可选关联车主，邀请查看私密留档（**只读，无需签字**）；
3. 车主 **授权公开** 后，脱敏沉淀案例与 GEO 内容（Phase 1 唯一车主主动确认环节）；
4. 提升门店透明度与咨询转化。

**Phase 2** 再启用：配件/方案车主确认、L2+ 阻断完工（见 `10_配件告知确认规则.md`）。

## **5. 六阶段编辑（Phase 1）**

按 `02_相册模板与节点规则.md` 分阶段上传：

1. 接车登记  
2. 检测诊断  
3. 维修方案与 **固定参考报价**（门店填写 `planAmount`，非平台成交价；私密查看均展示固定价）  
4. **配件告知**（标准类型 + 价格 + 凭证图 — **录入留档**，不发起车主确认）  
5. 施工记录（新旧件、包装、编码等）  
6. 完工交付  

**Phase 1 无阻断规则**：商家可在配件未「车主确认」时标记完工。  
**Phase 2**：L2+ 非原厂件未确认不可完工。

## **8. 邀请车主与公开案例**

### **8.1 邀请查看（私密）**

- 已填手机号：车主可在「我的 → 我的服务相册」查看；  
- 可分享小程序路径 / 码（P1）；  
- **仅查看留档**，不要求确认配件或方案。

### **8.2 授权公开（公开案例）**

- 须档案 `completed` 且至少 1 张有效图片；  
- **车主**在 user 端发起授权（勾选 + 留痕；Phase 2 电子签名）；  
- 脱敏工作台 → 平台展示审核 → 发布 H5 / 案例库；  
- Tag：**已授权 · 已脱敏 · 已审核**（按授权档位展示字段）。

商家 **不可** 代车主授权公开。

## **10. P0 验收标准（Phase 1）**

1. 商家可新建服务相册（门店 + 服务 + **车主扫码关联手机号**）；  
2. 可按六阶段上传图片与说明，可录入配件类型与价格；  
3. **不发起**配件车主确认，**不阻断**完工；  
4. 可邀请已关联车主查看留档；  
5. 车主可授权公开；公开前完成脱敏；  
6. 员工上传与关键操作有日志。

## **12. 与原「历史案例上传」关系**

Phase 1 **取消独立历史案例上传入口**。所有维修记录统一走 **服务相册**：

- 无车主、仅留档：须车主扫码关联后方可标记完工；  
- 需公开案例：须车主授权（②/③）；**不再支持**商家单方无车主冷启动公开。

原 `05_历史案例上传.md` 保留供考古与 Phase 2 参考，**新功能勿再引用**。
